Setting up Tags

Browse to Manage Learning > Manage Tags.

  1. Select Manage Tag Types.
  1. Select Add Tag Type.
  1. Name the Tag Type.
  2. Optionally, add a Description for the Tag Type.
  1. Enter a Name for each tag.
  2. Select the icon to add each tag.

As you add tags, they will appear in a list above the new tag entry.

  1. Select the icon to edit tags.
  1. When you are happy with your changes, select the icon.

  1. Select the icon to remove tags.
  2. Select Save Tag Type.
