Frequently asked questions

The Janison Cloud Learning System (CLS) enables you to deliver online learning and assessment simply and quickly across a variety of platforms and device. we have provided the information below to help you quickly find the answers to commonly asked questions.

Why does my site look different to the screenshots on the help site.

The CLS is highly configurable and therefore can look different for a variety of reasons:

  • Menu items can be hidden due to restrictions on the Role you are logged in with.
  • You may have a customised web page developed to meet your specific requirements. This page could be set as your home page.
  • The system comes with a set of default terminologies however most text can be changed to suit your requirements using String Resources. For example Tests can be renamed to Quizzes.
  • The Menus can be customised to suit your requirement for example the order can be changed or items added and removed.
  • You may not have certain plugins activated on your site.

 I've uploaded my new users but forgot to send the welcome emails. Is there another way to send welcome emails?

Browse to Users > Actions > Run Bulk Action. Select the action Send Welcome Email. You can use the filters to select the users you would like to send the email to.

 Why can I only see limited types of learning in Add Learning?

Add Learning uses the wizard to add learning assets. Only certain learning types are available here. To access all options browse to Manage Learning Assets > Add Learning Asset.

 Why are there menu items referenced in the help site that I cannot see.

Some menu items may not be visible if the plugin is not active on your site. In this situation please contact your Support Services team for further information.

 I have a catalogue item that I do not want to be displayed in Browse Learning.

Change the catalogue item to Make Private. You can control enrolments manually or by setting User Enrolment Rules.

 How long can CLS be idle for and how do I alter this?

Browse to Settings > Tenant Settings > Login > Session Timeout where you can adjust the time. The default is 30 minutes.

 The default CLS password specification doesn’t match our company’s internal specification. How can I change this?

At Janison we always recommend a strong password policy and recommend our default or stronger. If you’d like to change the default please browse to Settings > Password Settings > and select the items you want included in your password specification.

 The icons on my site are not displaying correctly.

This can occur when the site is accessed from a different URL to the usual one eg a redirect, custom URL or beta site. You can fix this by browsing to Settings > Styles and Images > and clicking the Save button.

 Is there an easy way to see what roles are assigned to a user?

View the user and expand the section System.

 A User account has been locked, how do I unlock it?
  • Navigate to Settings > User Settings.
  • Open the section List View and make the Actions field visible.
  • Now navigate to Manage People > Users.
  • Search for the required User but do not open the User.
  • You will find an icon of a lock in the Actions column.
  • Click the lock icon and the account will be unlocked.
What is the ideal size for an image loaded to the Login page?

There is no limit on the size of the image but we would recommend a minimum of 1100 x 600 px and 65kB to a maximum of 1440 x 900 px and 150kB. The bigger the image the longer it will take to load but the crisper it will look so you will need to find a happy medium.

 Why has a user has not received the Forgot my password email

There are a number of possibilities why this may occur:

  • The email has been directed to the user’s Junk E-mail folder.
  • The user has not entered a valid username. For security purposes the system returns a generic error message regardless of whether the user ID or password is incorrect.
  • Emails have been disabled for the site.
 What version of SCORM does the Janison CLS support?

The CLS supports SCORM 1.2.
