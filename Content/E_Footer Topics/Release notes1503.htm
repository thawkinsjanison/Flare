Release notes 1503 - release date 10th April 2015

New Features
Responsive UI means that the CLS has a great new look and feel! The layout will adjust according to the window’s size, meaning more information on screen and less wasted space. This also enables are far superior mobile experience.
Learning Plans allow a User to setup learning objectives, select relevant courses from a Catalogue in CLS or record External Learning and reflect on their experiences.
Search functionality in People plugin has been enhanced. Custom attributes can now be set to be visible in People
Previous searches are now 'remembered' after you refresh/forward/back/navigate away and return via Menu.
Added a new field for "Catalogue Item" reporting called "Average Rating", which is the average rating that users have given the catalogue within the system.
Multiple users can now be selected and added to a Group simultaneously.

Enhancements
Course developers can now preview their courses without enrolling.  They can also reset their test progress records using a reset button.  Previously they needed to un-enrol and re-enrol to do this.
Images can be uploaded as front page or thumbnails for course navigation, which will be resized on upload to reduce bandwidth overhead for end Users.
Ordered and filtered lists now keep any applied sorting when going in and out of full view.
Usability of Stimuli filters and results was improved.
A link to 'Message Templates' from 'Email Settings Notifications' was included to improve navigation.
Mutlichoice questions that have images as the distractors, in places where you can only show text (such as showing student feedback, will use the title text (or alt text if there's no title text).
This change improves search for catalogue items and tags by removing the ordering constraint of search phrases. It also introduces a stop word list which removes unnecessary search terms like "and". This means that you can now search for some like: "million max" and find "max has million's".
Bulk Imports will now display notifications so user can see these when "Enable System Notifications" is on.
Added a new report field "Organisation Identifier" to the User report.
Minor security enhancements.
A new option in Catalogue Item Settings now controls whether enrolled Users will be able to access that Catalogue Item once it has been archived.
