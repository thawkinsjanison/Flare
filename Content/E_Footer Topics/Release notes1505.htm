New Features and Enhancements for Release 1505

Completion certificates

Certificates are now accessible from the My Learning Transcript page.

See the document on certificates for further details.

Component Equivalency Groups

This feature allows groups of components to be created in a Catalogue Item, so that only a certain number of components within the group need to be completed.

See the document on Component Equivalency Groups for further information.

Additional notification available

An email notification is now available to advise managers if a learner fails their learning asset.

See the document on manager notifications for further information.

Discussion Forums

Discussion forums can now be embedded on custom pages.

See the document on Custom Pages for further information.

Learning Event Enhancements

  • Both administrators and learners can filter a list of Sessions using advanced search options including postcode.

  • Venue and Location creation has been streamlined and can be done on one screen, including when creating a Session.

  • Venues and Locations can be imported in bulk.

  • Both administrators (in several locations) and learners can hover over a list of Sessions and click to get a Session’s Venue and Location details including a google map.

  • Administrators can hover over a list of learners in a waiting list and get User details with the option to edit them.

  • Administrators can email Session attendees with a reminder notifications before it starts.

  • There is a new Learning Event option for Session waiting lists and another to enforce Session selection on learner enrolment.

Payment Gateway Enhancement

Previously we only allowed a single credit card and a single PayPal account per tenant. This functionality has been extended to allow multiple payment gateways per tenant. See the document on Payment Settings for further details.
