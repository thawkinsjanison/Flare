New Features and Enhancements for Release 1504

Note: These changes will be taking place in the first 2 weeks of August 2015.

Prerequisites

Pre-requisites can now be set between catalogue items.

See here for details.

Learning Event Session Enhancements

  • You now have the option to provide extra information to your attendees about particular sessions.
  • Each learning event session has its own summary page in the CLS.
  • When you have to make changes to a learning event session, you can choose to resend the email and calendar appointment to your attendees to notify them of the change.
  • You can now assign a provider to your learning event session.
  • The functionality to report on session providers and number of attendees etc. using Customisable Reports is now possible.

See here for details.

My Learning Transcript Enhancements

  • You now have more control over what appears on the learning transcript for your users.
  • PDF exports of the transcript can include Your organisation’s name and logo and a date range.

See here for details.

Email Notifications

  • Email reminders can now be set on learning with due and expiry dates.

See here for details.

Learning Plans Enhancements

  • There is now an option to Pre-populate a learning plan with Learning Plans entries.
  • Making Learning Plan entries mandatory or optional is now possible.

See here for details.

Reporting

  • Enrolment reports now have a 'subdomain' field, allowing you to see which site your Users are logging in from.

Test Questions

  • The new Save and add another button on User, Question & Stimulus screens makes adding multiple new objects easier!
