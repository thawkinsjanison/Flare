Inspect Learning Asset - Learning Event

A Learning Event is training that takes place when a learner User actually attends a venue, including a virtual venue (webinar). One event is made up of one or more Sessions which learner Users attend. Sessions are managed in Manage Learning>Manage Sessions, see Manage Sessions for details of how Sessions are managed and marked.

To inspect a Learning Event select Manage Learning > Manage Learning Assets and then click the name of the Learning Event you want to inspect. You will be taken to the Manage Catalogues section of the CLS where you can inspect this event.

  1. Tasks, Notes and Attachments can be viewed and added
  2. If the Learning Event is not linked to a Catalogue Item Users cannot enrol in it. Clicking Add it now takes you to New Catalogue Item where you can remedy this problem.
  3. The primary characteristics of the Learning Event are listed here including options chosen when it was created and whether it is active.
  4. The number of attendees in each of these categories is listed: Waitlisted, Attending, Attended, Interviewed, have RPL, Did not attend. The number of People on Waitlists is displayed prominently.
  5. Click the arrow next to Learning Event Sessions to view a list of the linked Sessions.

  1. Click the triangle to enter search terms to filter the list and display the results
  2. Click the arrow next to Competencies to view any Competencies attached to this Learning Event and whether awarded automatically
  3. Click the arrow next to Users who have registered interest to view this list

Click the arrow next to Recent activities if desired

  1. To edit this Learning Event click Edit Learning Event
  2. Click Add Session to add a Session to this Learning Event
