Search for Tests

The search feature allows you to quickly find a Test.

To search for a test select Design > Tests and Surveys.

  1. Click Search for Tests and Surveys and a search field will appear.
  2. Enter search field.
  3. There are two filter options: searching within specific module(s) or searching for Tests created, edited or flagged by you.
  4. Select options and click Search.
  5. The search results will appear under the Display Name column.
  6. To perform another search click Reset Search.
