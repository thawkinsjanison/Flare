Sequences

You may need to create a sequence of screens in a Test such that candidates see an introduction screen at the beginning and a screen at the end, that is different depending on whether they passed or failed.

In a multi-section Test you can create this by creating a series of different types of Test Sections, starting with one of type Introduction, and so on.

When you create a Test that does not have multiple Test Sections, a Sequence collapsible window appears on the Test's details page.
