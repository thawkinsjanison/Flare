Tests Introduction

In the CLS Tests are linked to Questions that belong to the same Module. Tests also include settings about how the Test is to be managed, e.g. how it is to be marked.

Tests can belong to Learning Modules or to Assessment Modules. A Test linked to an Assessment Module supports more question types and has many more options, such as how it is timed, ordered and delivered. This document covers Assessment Module Tests. Note that you can quickly create an enrollable Assessment Test using the Add Learning wizard which simplifies Test creation, slightly reduces the range of options available and creates a linked Catalogue Item.

When you create a new Assessment Test you specify the Assessment Module that it belongs to. However, the Test must be directly linked to an enrollable Catalogue Item in order to be presented to the User in a Test Player; merely belonging to an Assessment Module will not present it in the Test Player. By default the Questions in a Test are organised into a single section. However, you can create a Test with multiple Test Sections. Each Test Section is linked to one or more Questions and plays a different role in Test delivery, for example a Test Section that is only seen if candidates are exiting a Test that they have failed.

Note that the CLS caters for a wide range of uses: from a major Test development process involving teams of authors, editors and reviewers to a single Teacher with 25 minutes to create an automatically-marked quiz. To achieve this range of uses Test Designer settings are available under the Settings menu to customise how things work. Thus the functionality available in this section of the CLS is influenced by what has been selected in Settings > Test Designer Settings and everything described here may not be visible on your screen.

Note also that Tests and Surveys sit together in the left hand menu of the CLS, however they are quite different. A Survey is a tool used for gathering information and User feedback. Although it uses Questions, it differs from a Test as it is not assessable. Information about Surveys is at Surveys.

To access Tests navigate to Design > Tests and Surveys.

Click the Test tab to display Tests on the screen.
