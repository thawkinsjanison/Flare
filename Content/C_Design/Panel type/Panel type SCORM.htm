Panel type SCORM

The SCORM panel is used to display simple SCORM courses within the course player. This panel should be used for simple SCORM courses designed to enhance the learning aspects of a Janison course. The SCORM courses placed into a panel should be kept simple and not be concerned with storing progress within tests or questions.

To apply a SCORM to a course panel:

  1. Enter a Title for the Panel.
  2. Select SCORM from the Content Template option.
  3. Select your desired SCORM course from the SCORM Course option.

You will need to create a SCORM course beforehand to link it to a Course Panel.

  1. Select Edit metadata.
  2. Enter an optional Description for the SCORM course.
  3. Enter optional Text Alternative.
  4. Optionally, Enter a new Width and Height for the SCORM course within the Panel.
  5. Select Save.
