Panel type – Video

The video panel is designed to easily embed a video into the page without any code required. The recommended file type is mp4.  Other supported formats are m4v, f4v, mov.

To add a video:

  • Add a panel of Video type.
  • Upload a video using the button Select video.
  • Upload captions using the Select captions button.  Captions are optional.  The supported file types are 'srt, 'vtt, ttml and dfxp.
  • Select an image using the Select image button.  Before the video is played a black screen will appear.  If this option is used the uploaded image will appear instead.
  • Edit metadata provides options to add Text Alternative, video size and Captions.

If you’re finding that your video takes a long time to load on your site, then you can always use YouTube. YouTube is the video platform leader and is fast, free and easy.  It automatically manages aspects of the video such as compression and sizing.  If this method is preferred copy and paste the YouTube provided embed code into the Source code of Plain Text panel.

You will find YouTube Embed code under Share link.
