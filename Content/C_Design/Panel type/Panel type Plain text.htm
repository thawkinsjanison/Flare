Panel type – Plain text

The Plain Text panel is used to add text to a page. The Plain Text panel has an editor with standard editing functionality.

For advanced users the Styles tab can be utilised. For example if you wanted to control the size of the text for this page only you would use the following override.

.course-player #content-item {

       font-size: 14px;

}

The third tab Script is also for advanced users to add rich functionality to the page.
