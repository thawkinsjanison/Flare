Test Panel type –Poll Quiz

The Poll Quiz panel is designed for learner self-reflection rather than gathering survey information. It works with a Test that includes one or more Multiple Choice questions. Other Test question settings can be left as default. The Page is complete once the learner has answered the Questions.
