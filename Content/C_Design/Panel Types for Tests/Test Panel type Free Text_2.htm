Test Panel type –Free Text Review

The Free Text Review panel is not assessable and gives learners a read only summary of the response previously entered for that Question in a Free Text Panel. The purpose is self-reflection, as learners have the opportunity to reconsider their earlier understandings.

When selecting the Test from the drop-down menu, be sure to select a Test linked to a previous Free Text Panel.

In the Course Player, the learner sees in a read-only text box what s/he previously wrote. The Page is complete once the learner has answered viewed the Page.
