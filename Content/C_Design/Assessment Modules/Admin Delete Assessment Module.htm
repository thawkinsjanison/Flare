Admin Delete Assessment Module

To delete an Assessment Module navigate to Design > Modules and click the Module name. This takes you to its details screen

Click Admin Delete.

This is a serious and irreversible action that also deletes the entities that are only part of this Module. The following list for an example Module indicates the range of entities involved:

Because of the seriousness of this action, a second, high-level Admin User is required to authorise the deletion.

If this person clicks OK then the Module and all the entities that belong to it are permanently deleted.
