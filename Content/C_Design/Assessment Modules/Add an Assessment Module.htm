Add an Assessment Module

To add an Assessment Module, select Design > Modules and select Add Assessment Module from the Actions drop-down menu.

In the next screen you enter:

  1. The Discipline the Module will belong to.
  2. The Name and Identifier of the Module.

  1. You can add an optional Description if desired.
  2. If Workflows have been enabled and you want to use them, select the desired Workflow definition and settings. See the workflow document for details.
  3. Click Save Module to save the Module.

The next screen is the details screen for the newly-created Module. This enables you to manage various aspects of the Module, for example you can create a Test that belongs to the Module via this screen, see Inspect Assessment Module details for details.
