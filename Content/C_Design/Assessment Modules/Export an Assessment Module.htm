Export an Assessment Module

There are three options for exporting and copying Assessment Modules:

  1. Export a Module to a zip file. This activity is password-protected. A password is generated when the .zip file is created and it must be entered by a User attempting to import this file. Make a note of the password when you export the Module and keep it somewhere safe.

    2. Note that exported Modules can only be imported into the CLS and are not compatible with other Learning Management Systems.

  1. Secure Export a Module to download it as a token file.
  2. Deep Copy the Module and its related entities.

The first two options are performed via the inspection screen for an Assessment Module, see Inspect Assessment Module details. The third, Deep Copy, makes a copy of the Module together with all related entities, see Deep Copy Assessment Module
