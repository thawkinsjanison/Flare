Add bulk resources

This is useful if you want to load a lot of resources from your local machine.  To Add Bulk Resources navigate to Design > Resources.

Click on Add Bulk Resources from the Actions drop-down menu.

  1. Link the resources to a Discipline and Module.
  2. Add a tag to the resource by entering the tag name and clicking +.
  3. Click on Select files to browse your local drive and load the required resource.
