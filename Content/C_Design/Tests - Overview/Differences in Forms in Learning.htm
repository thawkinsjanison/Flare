Differences in Forms in Learning and Assessment Tests

There are several features available when creating a Assessment Test which are not available in a Learning Test.

Learning Test

Below is a form of a Learning Test. It has 5 sections:

  • The body.
  • Pass/Fail and Resit Settings.
  • Questions.
  • Response Analysis.
  • Recent activities

 Assessment Test

Below is a form of an Assessment Test. It has the 4 sections of the Learning Test has but in addition has sections for:

  • Timing Options
  • Player Skin Settings
  • Sequence

In addition to this you will notice a Warning that the Test has not been put in a Catalogue. As mentioned in the introduction, an Assessment Test is added directly to a Catalogue item. Learning Tests on the other hand are added to Panels of Pages in a Module. A Module needs to be linked to a course. The course is then added to a Catalogue item.
