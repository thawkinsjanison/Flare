Add a Question

To add a Question navigate to Design > Questions and select Add Test Question from the Actions drop-down menu.

The next steps depend very much on the type of Question you are creating, see Assessment Question Types for details.

The bottom of the screen also contains a section for notes on the marking rubric. Click Marking Rubric to open it and enter information in the text box.

Click Save Test Question to save this Question. Click and add another if you want to save this Question and then go straight to the New Test Question screen.
