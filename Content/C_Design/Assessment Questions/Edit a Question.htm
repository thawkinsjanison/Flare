Edit a Question

Once a Question has been saved, the Edit Test Question button is available.

To edit a Question navigate to Design > Questions.

  • Select the desired question from the Display Name list, or search for the question using the Search bar.
  • Select Edit Test Question.

You will be taken to the edit screen where you can change details as required. The fields available will be the same as the ones available when you created this question, see Assessment Question Types for details.

Click Save Test Question after editing it.
