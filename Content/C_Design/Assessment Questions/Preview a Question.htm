Preview a Question

Previewing a Question is an important aspect of Assessment Development. Questions can be previewed at any time however any Test Attempts for the Question while in preview mode will not be saved.

To preview a question navigate to Design > Questions.

Locate the Question you wish to view from the Display Name list, or search for the Question using the search bar at the top of the page.

Click its eye icon. The Test player will open in a new window:

Note that you can also preview a Question from its details screen.
