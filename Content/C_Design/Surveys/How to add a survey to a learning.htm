How to add a survey to a learning module

A survey needs to be added to a page of a learning module.  The user can either:

  • Create a new module and add a survey. 
  • Add a survey to an existing module. 

To add a survey to an existing module navigate to Design > Modules.

  • Search for the module using the search function or select the module from the Display Name list.

  1. Select Pages > New.

  1. Enter a title for the new page and click the tick.

  1. The Title and Identifier will automatically populate.
  2. Thumbnail – a thumbnail can be added but is not mandatory.
  3. Content Template – select full page from the dropdown menu.
  4. Activate New on the blank panel.

  1. Title – create a title for the panel.
  2. Content Template – select Survey Quiz from the dropdown menu.
  3. Test – the test section will appear.  Activate the dropdown menu and select the survey from the list.
  4. Select Save.

  • The survey is now linked to the module.  Edit and Unlink functions are available.
