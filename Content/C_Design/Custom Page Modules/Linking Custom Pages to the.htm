Inserting a Custom Page into the My Learning Smart Dashboard

The My Learning Dashboard is the page where learners access their catalogue enrolments. It is possible to add a custom page to the top of this dashboard. This provides the opportunity to include instructions or information for the Users in the form of videos, images, text or other media. Below is a screenshot of the My Learning Dashboard without the Custom Page.

Note: This option is only available if the Smart Dashboard plugin is active on your site. You will need to contact the Janison support services team to have this activated.

To add the page to the Dashboard you need to need to locate and update a String Resource with a key called JService/UiElements.DashboardIntro.

Select Settings > String Resources.

  1. You cannot search for this term but can easily locate the resource by clicking the Default Value heading. This will sort the resources by Default Value and those with this field blank will be listed first.
  2. Click the pencil icon to edit it.

  1. Paste <div jui-include="'/pages/Identifier'"></div> into Value, replacing identifier with the Identifier of your Custom Page as highlighted below.

In screenshot below the custom page has been added to the Dashboard. The Value of the String Resource is <div jui-include="'/pages/welcome'"></div>.

Note that you can also add custom pages to the My Learning Dashboard as separate tiles, see Pages in Custom Pages Modules for details.
