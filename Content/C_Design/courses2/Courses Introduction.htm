Courses – Introduction

In the CLS there are a number of different types of Learning Assets such as Courses, Trackable Documents and Learning Events. Learners are enrolled in a Catalogue Item that is linked to one or more of these Learning Assets.

Courses are containers for learning content (text, images, video, etc) and they are created within the CLS. However, Courses are actually connected to the learning content through links to Learning Modules that are in the same Discipline. It is the Learning Modules that actually contain the Pages (and Panels) that learners see in the Course Player. Panels can also be linked to Tests, Surveys and Questions that belong to the same Module.
