Add a Course

There are two ways to create a Course. The most direct is to select Design > Courses

You can also select Manage Learning > Add Learning Asset > New Course which brings you to the same screen.

  1. Select a Discipline from the drop-down list.
  2. Enter the Name and Identifier of the Course. These are mandatory fields.

  1. Enter any applicable Keywords for the Course.
  2. Further Information about the Course can be added but is not mandatory.

Thumbnails

Click the picture icons to select thumbnail images for the Course..

Available Modules

You will see a list of all of the available Learning Modules in the same Discipline as the Course. This allows you to view and link Learning Modules to the Course. Note that Modules need to have been created prior to linking them, see Learning Modules for details.

  1. Click the check box next to Show modules already in this course to show, in this window, the Modules already linked to the Course.
  2. Click the search box if you need to filter the list of available Modules.
  3. To link a Module click Add Module. This will update the list(s) of Modules in the Course.

Note that the Module names are clickable links to their details screens.

Competencies

If desired click the chevron to open the window and link one or more Competencies to the Course. These Competencies need to have been previously created, see Competencies for details. Users acquire these Competencies through the completion of the Course.

Save the Course

When you have completed all the desired fields click Save Course.

The next steps

If you are creating a Course from scratch the next steps are:

  • Create one or more Learning Modules and link them to this Course via Available Modules.
  • Add Pages and Panels to the Learning Module.
  • Optionally, create Tests and Questions that belong to the Learning Module and link them to one or more Test Panels.

And then if you want learners to see the Course in the Course Player:

  • Link this Course to a Catalogue Item and enrol learners in it.
