Search for Stimuli

The search feature allows you to quickly find a Stimuli. Search is only required when there are a large number of stimulus in the list.

To search for a stimuli select Design > Stimuli

  1. Click Search for Stimuli and a search field will appear.
  2. Enter search word.
  3. Click Search.
  4. The search results will appear in the Display Name column.
  5. To perform another search click Reset Search.
