Edit a Template

Unlike in other areas of the CLS where details can be changed, the Template edit page allows the User to view details but no editing functionality exists on global templates. This is because they are used by all clients and any changes that are made will affect a large cohort of people.  Any modifications made to the pre-existing templates will be made by the team at Janison. It is however possible to edit local custom made templates.

Select Design > Templates.

  1. Search for the required Template using the search function. Or
  2. Select the required Template from the Display Name list.

The following functionality exists on the edit page:

  • Export – the template can be exported as a zip file by clicking on Export.
