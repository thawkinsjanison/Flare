Add Content Layout Template

Content Layout Templates specify the layout of a Learning Module Page. For example, 30 + 70 over 100 means that the Page is split horizontally, then the top half is split vertically with a 30:70 ratio:

Each of these separate parts of the Page is a blank Panel that will subsequently be assigned a Panel when the Page is being developed, see Add a Page to a Learning Module for more details.

The system Content Layout Templates are shown when you are adding a new Page:

You can create your own Content Layout Templates if desired. To do so navigate to Design > Templates and select Add Content Layout Template from the Actions drop-down menu.

  1. Create a unique Title and Identifier for the new Template. The Title usually indicates the dimensions of the Page divisions.
  2. A brief Description about the Template can be added but is not mandatory.

  1. By default Programmer is checked. Click Designer to view a more a user-friendly mode with a single blank Panel to start with. When you move the cursor over the blank Panel you will see four + icons:

  1. You create new blank Panels when you click the + icons in a Panel. Each of the four icon adds a blank Panel is a different way.

    2. For example, if you click the top + icon indicated by number 4 (above) you will create a new blank Panel above the starting Panel, thus making a layout template of 100 over 100. If you click the bottom + icon you will make a new blank Panel below the starting Panel.

    Clicking the + icon at the left hand side makes a new blank Panel in the same row on the left that shares the space with the starting blank Panel in a 1:1 (or 50 + 50) proportion. You can change these proportions by selecting values from the drop-down list, as indicated below:

  1. Continue adding blank Panels and adjusting proportions until you have the desired configuration. These can be quite detailed. In the example above, a Panel was added above the starting Panel (click the top + icon) and then another added to its row (click the left + icon) with a 1:1 (or 50 + 50) proportion. Then another blank panel was added below the starting Panel (bottom + icon) and another blank panel added to its row with a 3:7 (or 30 + 70) proportion. Note that as soon as you have more than one blank Panel red x icons appear that allow you to delete the blank Panel.
  2. Once you have created your Template scroll down and click Save Content Layout Panel.
  3. When the Template has been saved it will be presented as an option when creating a new Page in a Learning Module. In the example below, the Template above has been used for the Page's layout:

It is important to note that this new Content Layout Template is only available for Pages within your local system. It can therefore be edited and deleted if needed.
