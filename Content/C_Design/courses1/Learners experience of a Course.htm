Learners experience of a Course

Learners are enrolled in a Catalogue Item that is linked to a Course and they click on the Catalogue Item’s tile in their My Learning screen to get it going:

  1. If there is only one Course in the Catalogue Item and it has one Learning Module, they go straight to the Course Player showing the first Page of the Module:

  1. If there is one Course in the Catalogue Item and it has multiple Learning Modules, they go to a screen showing tiles for each Module. They click on a Module’s tile and go to the Course Player showing the first Page of this Module:

  1. If there is more than one Learning Asset linked to the Catalogue Item (technically, linked to a Program), then they go to a screen that lists the Learning Assets and enables them to launch them individually. Clicking a Course takes them to options 1 or 2 above, as appropriate.
