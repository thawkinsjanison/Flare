Available Question Types

The CLS provides a number of question types for Learning Modules:

  • Match
  • Multiple Choice
  • Multiple Choices
  • Sequence
  • Short Answer

There is a much larger number of question types for Assessment Modules.

In this section you specify which of these types of questions are available for Assessment Questions, including where and how.

Click the Available Question Types collapsible window. There is a row for each type of Question. For simplicity this screenshot does not show them all:

There are several column headings and they do not all apply to every type of question Where they are relevant to the question type, a checkbox is present.

  • For Tests – if checked this question type is available in Assessment Tests.
  • For Surveys – if checked this question type is available in Assessment Surveys
  • For Composite Questions– if checked this question type is available in a Composite Question type, which must itself be enabled.
  • Explain answer – if checked a box appears in the Test Player for this question type where learners need to enter an explanation for their answer.
  • File upload – if checked a section enabling the attachment of files to the Assessment Question will appear for this question type.
