Available Test Section Types

Test Sections are linked to Assessment Tests and are themselves linked to Questions, see Test Sections. Here you will also find a description of different types of Test Sections, the role that they can play in the Test Player and the types of Questions that can be linked.

Click the Available Test Section Types collapsible window. There is a row for each type of Test Section and a column where you can check which ones will be available in your system.
