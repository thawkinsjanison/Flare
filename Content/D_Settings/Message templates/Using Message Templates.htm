Using Message Templates

Message Templates in the CLS are simplistic and very self-explanatory, but there are quite a few things that we need to know to use Message Templates to their full potential.

  • Message Templates is located in the Settings menu, you will find this menu under the icon.
  • All default Message Templates have the scope as Global.

  • The templates with the scope set as Global cannot be edited and must be recreated as a Local Message Template.
  • Select the template you wish to edit and click the Actions button.

  • Choose Create Local Message Template this will create a clone of the template you have selected so it can be amended.
  • The templates are very flexible with the amount of formatting. It includes the regular features of a WYSIWYG editor (what you see is what you get). Features include, text formatting, links, images and HTML tokens.
