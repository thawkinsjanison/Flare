Further Information and Notes
  • Like most areas of the CLS, Message Templates can be exported and Imported between tenants. To export, select a Template > Actions > Export Template. To import a Template, hit the Import Template button on the main Message Template page.
  • Message Templates cannot be deleted, they can only be set to Disabled or Do Not Use.
    If a local Message Template has not been created for the Global Template you wish to disable, then do so.
    From here, select that Template > navigate to Actions > Edit Message Template and under the Status drop-down select Disabled – this will ensure the email is not sent or Do Not Use – this will fall back to the tenant or Global version of the template you are trying to disarm.
  • To Customise a Message Template, you must first Copy it as a Local Message Template.
