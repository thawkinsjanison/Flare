Custom CSS

It should be unnecessary to do this once the tenant has been set up, however is possible to enter custom CSS and override the style of the CLS in targeted areas. Note that you need to know what you are doing here and to be prepared to maintain it in future CLS updates when things may change.

To enter custom CSS select Settings > Style and Images and click the Custom CSS tab.

An example is:

  • change the colour of links in the Footer to blue:

footer a {

  colour: blue;

}

There are many free tutorials online that can help you learn about CSS, start by clicking here.
