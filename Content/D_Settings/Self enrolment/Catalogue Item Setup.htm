Catalogue Item Setup

After you have decided which options you want to allow you can start setting up your individual Catalogue items.

Move back to the icon and from the drop down select Manage Learning.

  • Under Manage Learning choose Manage Catalogue.

  • Find the Catalogue item you wish to apply the new Enrolment Rule to.
  • Choose Actions and Edit Catalogue Item.

  • Find the Self Enrolment Settings dropdown tab for that Catalogue item.
  • We can see that on this item the Enrolment Method is set to ‘Default’ which I have set to ‘Self-Enrolment (Free)’.

  • From the dropdown select the new enrolment rule you wish to override the default.

  • Save Catalogue Item.

Note: This can be applied to all catalogue items if desired; for variety in Self Enrolment methods.
