Setting enrolment options to appear at catalogue level

If you want a variety of different enrolment settings you can set Enrolment rules at individual catalogue items. An example of this is if you have a mixture of free courses and paid courses. You will need to Allow this setting under Self-Enrolment Settings.

  • Navigate to the Settings menu.
  • Select Self Enrolment Settings.

Tick ‘Allow Enrolment Method override per Catalogue Item’. This means that when you select an option at the Catalogue level it will override the default setting.

  • You will need to tick off which options you want to be allowed to set at the Catalogue level.
