Self Enrolment Introduction

Self-Enrolment Settings control the way learners enrol into Catalogue Items. For example Catalogue items can be set to allow enrolments without restriction or they can be set to require a payment to be made to access the learning. Other options are possible.

There are two steps involved to make this feature active:

  • Make the types of enrolment methods available in Self enrolment Settings. This step will only be required once.
  • Set up the enrolment type in the Catalogue Item.

To access the setting navigate to Settings > Self Enrolment Settings.
