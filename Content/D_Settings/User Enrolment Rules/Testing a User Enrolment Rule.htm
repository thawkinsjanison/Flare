Testing a User Enrolment Rule

To ensure that you have the rule set up correctly and also to check who will be affected by the rule or to run the rule, a Test feature is available.

To test an enrolment rule:

  • Navigate to Settings > User Enrolment Rules.
  • Select the required enrolment rule.
  • Click Test Action Rule.

A page will open with 3 tabs:

To Be Enrolled – This will list the Users who will be enrolled when the rule is run.

Already Enrolled – This will list the Users who have already been enrolled when the rule was previously run.

All – Includes all Users.

When you click Run Now, All Users listed under To Be Enrolled tab will be enrolled. If you do not click Run Now, the rule will enrol the Users when the system runs the Daily Check or when the User Registers, depending on what is set in When should the rule apply?
