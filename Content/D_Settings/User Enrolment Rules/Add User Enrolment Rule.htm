Add User Enrolment Rule

To add a rule navigate to Settings > User enrolment Rules and click the Add User Enrolment Rule button.

  1. Enter a relevant name.
  2. The Identifier will auto populate when accessing this field.
  3. The description is optional but is helpful for explaining the purpose of the rule.
  4. The rule must be set to Active to run and can be deactivated if no longer required.
  5. The primary contact will receive an email after the rule is run. The message template used for this email is ActionRule-UserEnrolmnentRuleRun.
