Block Browser

This option allows you to control whether some of the older browsers can log onto the CLS.

Allow Legacy Browsers to Logon

This option does not restrict any browser.

Prevent Legacy Browsers from Logging On

This gives you the choice to prevent any or all of the listed browsers from logging onto the CLS. When selected an editor will appear with the option to enter a custom message which will appear to Users attempting to log on with any of the blocked browsers.
