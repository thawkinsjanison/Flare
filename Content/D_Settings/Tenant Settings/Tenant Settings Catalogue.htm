Tenant Settings – Catalogue Item Settings

Enable Catalogue Items for Guest Users

Note: This feature should not be used without consulting with Janison support services as it is only intended for very specific requirements. As no progress will be accessible for the user it is recommended that the Instalaunch link feature is used instead. Details of this feature can be found in the document on Manage Catalogue.

Enable prerequisites within catalogue items

Prerequisites are items that must be completed before enrolment in a particular learning entity is permitted. If checked a Prerequisite column will appear in the Program Components section of the Catalogue Item. Further details can be found in the document on Manage Catalogue.

Enable Component Grouping

This option allows Learning Assets to be grouped in the Components section of Catalogue items. Please see the document on Manage Catalogue for further details.

Enable Component Equivalency Groups

This setting will allow groups of components to be created for a Catalogue Item, so that only a certain number of components within the group need to be completed.

Enable custom templates for Catalogue Item presentation

The way catalogue items look and behave can be customised by linking Program Templates to the catalogue. This enables a drop-down in the catalogue to select the required template. Further information on this can be found in the document on Templates.

Enable Additional Enrolment Info in the middle of a program

This option allows you to include a set of questions in a catalogue item or program which will be presented to a User during enrolment in a catalogue item. The questions are Custom Attributes grouped into a set. For further details on this please refer to the document on Custom Attributes.

When a User clicks the Enrol button the questions will be presented to them.

Allow addition of Resource Sets into Catalogue Item

This will add an option to the Components drop-down in a Catalogue item which will give you the ability to add Resource Sets to the Catalogue.

Cascade Catalogue Archive to Learning Assets

If this option is enabled when you archive a Program Catalogue Item, all of its Learning Assets and Components will also be archived.

Show Identifier in Search

If this option is enabled the Catalogue Item’s ID will be shown when searching.

Prevent users from accessing Catalogue Items when archived

This setting will prevent enrolled Users from accessing Catalogue items whose status is Archived.
