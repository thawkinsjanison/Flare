Tenant Settings – Certificate Templates

If Certificates have been enabled for a Catalogue Item they are generated whenever a User successfully completes it. A certificate is based on a PDF template. The CLS populates this PDF template, that is, it enters the appropriate data into the matching fields in the template. Thus, the template can be customised for each User, for example with the addition of the User’s name and their completion date.

Further information on how to create and use certificates can be found in the document on Certificates.

A field Has Certificate exists in the catalogue item and if it is checked, fields to configure the Catalogue certificate will appear.
