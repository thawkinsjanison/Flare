Browse Learning - Recommended Content

This feature gives learners the option to Recommend Catalogue Items to other learners.

Learner experience of Recommended Content

If enabled learners will have access to a Recommend to a colleague button. This feature works in conjunction with the Show Tabs setting in Catalogue Appearance.

If the Recommend to a colleague button is clicked an email pop-up will appear with the following fields.

  1. A Recipients drop-down where target learners can be selected.
  2. Once selected a list of recipients which can be removed using the icon.
  3. The Subject of the email. This field can be edited.
  4. The Message of the email. This field can be edited.
  5. The Send Email button sends the email.
