Using Message Templates

  1. Message Templates are located under the Settings menu.

    2. Select Settings > Message Templates.

  2. All default Message Templates have the scope as Global.

  1. Templates with the scope set as Global cannot be edited.
  2. To edit a Message template make a local Message Template:

    3. Select the template you wish to edit and click Actions .

  1. Choose Create Local Message Template from the drop-down list. This will create a clone of the template so it can be edited.
  2. The template has a WYSIWYG text editor (what you see is what you get). Features include, text formatting, links, images and HTML tokens.
