Further Information and Notes

  • Like most areas of the CLS, Message Templates can be exported and imported between tenants.
  • To export, select a Template then select Actions > Export Template.
  • To import a Template, select Import Template on the main Message Template screen.
  • Local Message Templates cannot be deleted, they can only be set to Disabled or Do Not Use.

    • Select the template, navigate to Actions > Edit Message Template and under the Status drop-down list select Disabled. This will ensure the email is not sent.

    Alternatively select Do Not Use. This will fall back to the tenant or global version of the template you are trying to disarm.
