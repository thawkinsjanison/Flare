Tokens and how they work

HTML tokens are an important feature of Message Templates. They are used to insert information that is held in the system into the email.

For example, the template Discussion – InappropriateBehaviour is used for the email sent when people report an inappropriate comment or post in Discussion Forums.

You can see below that the token [User.FullName] is in the first line.

When the email is sent [User.FullName] is replaced by the full name of the User who reported the inappropriate comment, for example John Doe.

Thus, the first line of the email message will be - Person reporting inappropriate behaviour – John Doe

Similarly, the second line of the message will be- Email address of person reporting inappropriate behaviour – Jdoe@email.com.au

To see how a Message Template would appear in an email, you can select Send test Email to send a test email to a nominated email address.
