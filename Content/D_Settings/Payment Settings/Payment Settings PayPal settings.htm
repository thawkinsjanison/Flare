Payment Settings - PayPal settings

Below is the flow process for a learner using Paypal as a payment method.

To complete the PayPal form you will need to obtain your Authentication Settings from PayPal. You will need to obtain the following information:

  • API Username
  • API Password
  • Signature

To obtain these details log in to your PayPal account via their website and navigate to My Account > Profile > API Access > View API Signature.

Clicking View API Signature will open a page with credentials for the Paypal Settings form.

  • Enter the PayPal credentials into the form and click Save Settings.

Note: If the Use Sandbox check box is ticked PayPal Payments will be put into test mode meaning that all transactions will go to the PayPal sandbox site. You can see more information here.
