Payment Settings - GST settings

This section allows for the configuration of GST if it is going to be included in the costing.

  1. None – refers to no GST requirements.
  2. Fixed – Select if you require a fixed GST rate.
  3. Variable – If the rate needs to be variable activate the radio button then type in the different rates in the columns below. For example the variations could be Australia 10%, New Zealand 15% etc. Each rate requires an Identifier. Click Add to save the data, Edit to make a change or click the icon to delete any unwanted data.
  4. Custom Attribute – custom attributes can be selected from the drop-down list but need to be set up prior to completing this step. For more information about this please refer to the document on Custom Attributes.
  5. Select Save Settings.
