List view

This refers to the details that are visible on the User page under the Search menu bar highlighted in the screenshot below. This page is accessed by navigating to Manage People > Users.

The setting options are as follows;

  1. Visible – check the boxes beside the fields you would like to appear on your registration form.
  2. Batch Edit – this is only applicable for custom attribute fields. It allows for information about that custom attribute to be updated for more than one User. There are a few steps involved. These include;

Create custom attribute – please refer to Custom Attributes.

Navigate to Settings > User settings > List View, locate the newly created custom attribute from the list of fields. Check the Batch Edit box.

The next step involves selecting the Users that you wish to add the custom attribute to.

Browse to Manage People > Users and check the box beside each User.

Click Set First Aid Certification. This button will be unique to the custom attribute.

A pop up box will appear. It is important to be specific here as the exact wording will appear on the Users Edit page.

To check the success for your Batch Edit navigate to Manage People > Users and select a User that has the Custom Attribute added to their profile. Here is an example.

  1. Order – drag the arrows to change the order the fields will appear on the form.
