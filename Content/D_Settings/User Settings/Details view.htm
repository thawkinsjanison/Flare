Details view

This refers to information available on the Users details page. It is accessed by clicking on the specific User. Navigate to Manage People > Users and either select the User from the Display Name list or Search and select the User. The first part of this form has 5 options that can be checked. These include;

Show Styled Summary – changes the layout and styling of the top part of the Users details page. The first screenshot shows the details page with this option unchecked. The second screenshot has the styling checkbox checked.

Show Section Completion – includes a section on the details page that will list all learning completions.

Show Section Purchases – includes a section on the details page that will list all purchases made by the User.

Show Section Activities – activates two new sub sections detailing Activities performed by and on the User.

Show Section User Roles – will include a section on the details page about the role of the User.
