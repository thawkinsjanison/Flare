Edit form

This refers to the information required to be filled out when a new User is being added to or edited in the system.

Navigate to Settings > User Settings > Edit form.

  1. Visible – check the boxes beside the fields you would like to appear on your registration form.
  2. Mandatory – checking this box makes the field mandatory. This will be recognisable by a red asterisk beside the field on the form.
  3. Capitalisable – if this box is checked any data entered into the field will automatically be capitalised.
  4. Order – drag the arrows to change the order the fields will appear on the form.
