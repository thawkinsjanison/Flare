Add a Venue

To add a Venue select Settings > Venues > Add Venue.

  1. Enter the Name of the Venue
  2. Enter its Capacity i.e. the number of people that it can hold.

  1. To select an existing Location for this Venue, click and select it from the drop-down list. Note that after you have done this, the option at 4 changes and you can edit the Location at the same screen by clicking Edit location now, see Create/Edit location now.

  1. To create a new Location in this screen click Create location now. This will open a blank Location form where you can enter the details, see Create/Edit location now for instructions.
  2. Click Save Venue

Create or Edit Location Now

This form allows you to create a new Location or edit an existing one. You reach it from within the creation or editing of a Venue.

The first three fields must be filled in. Address details are essential if you want map and location details to be available for a Session at this Location.

  1. Enter or Edit its Name
  2. Enter or Edit the Identifier
  3. Enter or Edit a Description
  4. Enter or Edit the Address details and other information as required.
  5. You can click Cancel location create or Cancel location edit if you do not want to proceed with creating/editing this Location.

Cancelling the creation of a Location will remove the name of the new Location from the Venue’s Location box.

Cancelling Location edit will leave the Location unchanged.

Click Save Venue. This will also save the created/edited Location.
