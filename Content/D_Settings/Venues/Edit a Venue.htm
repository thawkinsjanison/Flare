Edit a Venue

To edit a Venue navigate to Settings > Venues and click the desired Venue. Click Edit Venue from its details screen.

  1. You can change its Name
  2. You can add or edit its Capacity.
  3. You can change the Location linked to the Venue by selecting another Location from the drop-down list or by clicking X to delete the current link to a Location.
  4. If you have no linked Location you can click Create location now to create a new Location and link it to this Venue.

If you already have a linked Location, this button becomes Edit location now and allows you to edit the Location’s details on this screen, see Create or Edit Location Now.

  1. Click Save Venue when edits have been completed.

Note that clicking Save Venue also saves any changes to the Location too.
