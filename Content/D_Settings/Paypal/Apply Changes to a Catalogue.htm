Apply Changes to a Catalogue Item
  • Select Manage Learning > Manage Catalogue
  • Select the Catalogue Item you want to have PayPal setup on > Edit the Catalogue Item

  • Ensure this Catalogue Item has a price greater than $0
  • Scroll down, under Self Enrolment Settings, set Enrolment Method to Self-Enrolment (Paid).
  • Set Payment Method to PayPal

  • Save Catalogue Item.
  • When a user enrols in this Catalogue Item, they will be presented with a Self-Enrolment wizard.
