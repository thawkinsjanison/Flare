Setup with Payflow Pro

The PayFlow Pro gateway provides checkout solutions for novice and advanced use. It provides a new and alternative option for CLS Administrators to handle payment transactions for their catalogue items.

The Payflow Pro gateway is a plugin that needs to be switched on at Global level by Janison. Contact Janison to ensure the Payment Tenant Specific Plugins has been switched on at the global level.

After this has been done, confirm that Credit Card Payment is visible in the settings menu. If it isn’t, you may have a custom menu; in which case you will need to follow these steps:

Select Settings > Menu

  • If the Custom Menu box is not empty, add the following line to your Custom Menu (in the correct location) <knownMenuItem id="credit-card-payment" caption="Credit Card Payment" />
  • You can find its corresponding location in the Built In box.

  • Copy and paste the ‘Cred Card Payment’ line into your Custom Menu and Save.
  • Navigate to Settings > Select your new Cred Card Payment option.
  • Select Payflow as the gateway type.

  • You will need to obtain these details from PayPal in order to complete this form:
  • Host Address
  • User
  • Vendor
  • Partner
  • Password
  • Ensure Allow PayPal Payments is ticked. 
  • Expand the PayPal section and confirm Use Payflow Authentication is ticked.
  • Note: The Use Payflow Authentication checkbox will only appear when the user, vendor, partner and password fields have been filled.
  • Save your settings.
