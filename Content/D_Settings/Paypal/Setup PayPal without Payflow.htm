Setup PayPal without Payflow Pro

PayPal Express is a payment option offered to clients that does not require a payment gateway, it is all done via a PayPal seller account which you can setup with PayPal.

To use PayPal with your PayPal seller account you will need to obtain these details from PayPal:

  • API Username
  • API Password
  • Signature
  • To obtain these details, log in to your PayPal seller account on the PayPal website and navigate to
    My Account > Profile > API Access > View API Signature.

  • This will show your credentials

  • Once you have your PayPal credentials, Log into the Janison CLS site and go to Settings > Credit Card Payment then expand the PayPal section.

  • Enter your PayPal credentials and click Save Settings.
  • Note: If you check the Use Sandbox option, this will put PayPal Payments into test mode and all transactions will go to the PayPal sandbox site. You can see more information about this at https://developer.paypal.com/webapps/developer/docs/classic/lifecycle/ug_sandbox/
