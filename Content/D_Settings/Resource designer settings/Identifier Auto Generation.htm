Identifier Auto Generation

This section allows for the configuration of the format for auto generated Identifiers for both Resources and Stimulus.

Before any details can be added the Resource or Stimuli Identifier Auto Generation check-box must be ticked. Until this occurs the other field boxes are grey and will not allow any information to be added.

To add Identifier Patterns:

  • Select a Token from the Choose Token drop-down. This will be added to the related Stimuli or Resource Identifier Pattern field.
  • Select additional Tokens if required.

In the screenshot above two tokens have been selected for Resource identifier Pattern, [Discipline.Identifier] and [Module.Identifier]. One token has been selected for Stimuli Identifier Pattern, Number (8 digits).

  • Save Resource Designer Settings.

Note: It is also possible to add free text to the Resource or Stimuli Identifier Pattern field. For example if you added 'a' to the field all new Resources and Stimuli identifiers would be prefixed with 'a'.

Auto Generated Identifiers are Read-Only

If this option is checked the Identifier field will not be editable by Users creating Resources or Stimuli.
