Account Activation / Deactivation

Individual accounts or groups of accounts can be set to active or inactive when required using the Deactivate Accounts or Activate Accounts features of the Bulk Action tool.

  • From the main menu select Manage People > Users.
  • Select Run Bulk Action From the Actions Menu.

  1. From the available actions, select Deactivate Accounts or Activate Accounts.
  2. Specify the filters to target the appropriate users. For details on how to use the filters please see the documentation on building filters.
  3. Enter your password.
  4. Click the Deactivate Accounts or Activate Accounts button and a confirmation popup will appear.

Select Yes on the confirmation pop-up if you would like to proceed.

There will be a note on how many users will be affected by the action.

Once the action is performed, you will be directed to the Scheduled Job Details page with information on the status of the action. Further details can be seen when clicking the Details link under Results.
