Search for group types

The search feature allows you to quickly find a Group Type.  Search is only required if there are a large number of Group Types in the list.

To search for Group Types select Manage People > Group Types.

  1. Click Search for Group Types and a search field will appear.
  2. Enter search word.
  3. Click Search.
  4. The search results will appear in the Display Name column.
  5. To perform another search click Reset Search.
