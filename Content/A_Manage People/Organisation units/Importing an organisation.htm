Importing an organisation unit

Note:  A CSV should be created prior to importing an Organisation Unit.

Select Manage People > Organisation Units.

  1. Click Import Organisation Units.

  1. The CLS provides a sample excel template for your convenience.  To use this click on Template, populate the spreadsheet with the data, save, and select from your local browser. Or
  2. Search for and select your required file from the local browser.
  3. Click Upload.

Search for organisation unit

The Search feature allows you to quickly find an Organisation Unit. Search is only required when there are a large number of Organisation Units in the list.

To search for an Organisation Unit select Manage People > Organisation Units.

  1. Click Search for Organisation Units and a search field will appear.
  2. Enter the search word.
  3. Click Search.
  4. The search results will appear in the Display Name column.
  5. To perform another search click Reset Search.
