Searching for a Position

The Search feature allows you to quickly find a Position.  Search is only required when there are a large number of Positions in the list.

To search for a Position select Manage People > Positions.

  1. Click Search for Positions and a search field will appear.
  2. Enter search text.
  3. Click Search.
  4. The search results will appear in the Display Name column.
  5. To perform another search click Reset Search.
