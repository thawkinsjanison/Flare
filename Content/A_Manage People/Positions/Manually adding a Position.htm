Manually adding a Position

Select Manage People > Positions > Add Position.

Complete the Position form details:

  1. Enter an Organisation and Organisation Unit
  2. Enter a Name and Identifier
  3. Click in the Reports to box and select a Position from the drop-down list if desired.

Note that the system does not list any positions in the drop-down list if their selection would result in a circular reporting hierarchy.

  1. Check the box to make the Position Active.
  2. Enter a Description if desired.
  3. Click Save Position.

Note that Custom Attributes can be added to Positions. If they have been added they will be seen in Edit mode under the heading Extended Information.
