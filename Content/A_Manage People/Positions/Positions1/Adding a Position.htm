Adding a Position

There are two ways to add Positions:

  1. Manually, using Add Position.
  2. Importing them using a CSV file.

Note that depending on how Positions were set up you may need to find the Position functionality through an Organisation Unit, see Setting Up. If User-to-User manager relationships has been selected, click Positions from the Organisation Unit

This takes you to the Organisations screen where you can create, edit and manage the Positions in that Organisation.
