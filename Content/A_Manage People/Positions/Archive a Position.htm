Archive a Position

The archive feature removes the Position item from the list of active Position items. This assists in keeping the list to a minimum. The archived item can be accessed in the Archived tab. Generally speaking, archived items are ones that are no longer required.

There are two methods for archiving a position. These are:

  • Via Edit Position.
  • Via Position List.

Archive via Edit Position

Locate the Position and edit it.

  1. Uncheck the Active Box by clicking it.
  2. Click Save Position.

Archive via Position List

Select Manage People > Positions.

  1. From the list of Positions click the check box to the left of the Position you want to Archive.
  2. Click Archive.
