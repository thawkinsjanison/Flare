Search for People

The search feature allows you to quickly find a person. In this area of the system there could potentially be thousands of active Users. It is possible to customise the search filters to target specific groups of people and enable a more powerful and precise search.

To search for a person select Manage People > People.

Note: if searching using one of the first three search options click on Advanced to hide the more complex options otherwise the search will not complete.

  1. Search for User – type in part or all of a User’s name click Search.
  2. Search for Location – type in the Organisation to search through and click Search.
  3. Search for Catalogue – click in the text box and a drop-down menu will appear displaying all catalogue items. Scroll through and select the catalogue item before clicking Search.
  4. Building filters - You can set up rules to construct a more precise search. These rules are called filters.  For detailed step by step instructions about this functionality please refer to Building Filters.
  5. At any time Reset Search can be selected to start the search process again.

Search results will appear in a list below the Search box.
