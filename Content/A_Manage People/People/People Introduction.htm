People - Introduction

The People section of the system is more than just a list of all active Users.  It is a place to go to connect with others that may be enrolled in the same Learning Assets, seek out experts for advice, link to discussion forums and report any inappropriate activity.

To access People navigate to Manage People > People.

Each active User has a tile with some details visible.  The details displayed here can be configured in User Settings.  More on that later.

  1. Click on the magnifying glass to expand the profile and find out further details.
  2. User’s name and the Organisation the User belongs to.
  3. Click on the exclamation mark if there is a need to report any inappropriate activity.
  4. The blue triangle highlights Administrators.
  5. Blue speech bubbles indicate the User has been involved in a discussion forum. Click on the icon to visit the discussion forum. If the speech bubbles are grey the User has not yet contributed to a discussion forum.

It is not possible to edit any details from this section. If a User wants to make any changes to their profile they can click their name at the top left and edit from there.
