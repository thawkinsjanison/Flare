Adding fields to the People tile

It is possible to customise the information available on the People tile. This is done by creating Custom Attributes linked to the Users object. There are three main steps in order to complete this process:

  1. Create a Custom Attribute.
  2. Set the Custom Attribute to Visible in the Public Profile.
  3. Populate information into the new User field.

Create Custom Attribute for People tile

Navigate to Settings > Custom Attributes and click on Add Custom Attribute.

  1. Create a Name and Identifier for the Custom Attribute.
  2. A brief description about the Custom Attribute can be added but is not mandatory.
  3. Select the Type of Custom Attribute from the drop-down menu. You can find details about the Type in the Custom Attribute document.
  4. Set a maximum length.

  1. Select Users from the Object drop-down menu.
  2. The Caption is the exact wording that will appear on the People tile.
  3. Check the Mandatory box if you require the population of the field. The field will be marked with an asterisk.
  4. If Inactive is checked the Custom Attribute will become inactive although will still be present in the system.
  5. The button in the Actions column will delete the Object and Caption.
  6. Select Save Custom Attribute.

Set the Custom attribute to Visible

The custom attribute needs to be activated in the Public Profile before it will appear on the People tile. To do this Browse to Settings > User Settings and open the Public Profile section. Find the new custom attribute in the list and check it as Visible.

Note: The cross highlighted above allows the fields to be re-ordered. Click and drag the field as required. The order in which they are displayed in this list will be the order they are displayed on the People tile.

It is important to understand at this point that even though there are Custom Attributes set up and checked as visible the information will NOT display on a People tile unless the User has filled out the information on their personal profile page.

Populate information into the new User field.

Navigate to Manage People > Users.

  • Search for and select the required User.
  • Click on Edit User from the Actions drop-down menu.

  • The New Custom attribute will be available in the list of details. Update the information as required.
  • Select Update Administrators Profile.

The expanded view of the People tile will now display this new information.
