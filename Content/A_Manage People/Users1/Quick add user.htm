Quick add user

This option enables the addition of multiple Users at a time, as long as they all belong to the same Organisation or Organisation Unit.

Select Manage People > Users > Quick Add Users.

  1. Select the Organisation and Organisation Unit the User will belong to from the dropdown.
  2. Check Send Welcome Email to User if this option is needed.
  3. Fill out users details.  Click in the Username box to add to the list of Users.
  4. A User can be deleted from the list by clicking the - button under Actions.
  5. Click Save Users.
