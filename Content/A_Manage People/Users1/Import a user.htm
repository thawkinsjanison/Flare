Import a user

Users can be added to the site in bulk using a spreadsheet import.

Select Manage People > Users > Import Users from the Actions drop-down.

  1. The CLS provides a excel template that you can use to populate with your users. To download the file select Template. The Supported Attributes tab of the Template provides the mandatory fields and other information about the requirements for the data fields.

  1. Select your import file from the local browser.
  2. Optionally check any of the boxes in the list of additional options if required.  For more detail about these options hover over the question marks. 
  3. Enter Email Address.
  4. Select Import and email result.
