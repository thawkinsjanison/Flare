Import a user

It can be useful to import information into the system via a spreadsheet to limit the amount of data input required. 

Select Manage People > Users > Import Users from the Actions drop-down menu.

  1. The CLS provides a sample excel template for your convenience.  To download the template click Template. The Supported Attributes tab of the Template provides the mandatory fields and other information on the requirements for the data fields.

  1. Select your import file from the local browser.
  2. Check the box beside the list of additional functionality if required.  For more detail about these options hover over the question mark. 
  3. Enter email details.
  4. Click Import and email result.
