Archive a user

The purpose of the Archive feature is to remove a User from the list of Active Users.  This assists in keeping the list to a minimum. The Archived User can be accessed from the Inactive tab.  Generally archived items or Users are no longer required.

There are two ways to archive a User.  These are:

  • Via the Edit User.
  • Via the User list.

Archive a user via edit user

Select Manage People > Users.

  1. Search for the User using the search menu.  Or
  2. Click on the User from the Display Name list.

  1. Click Edit User from the Actions drop-down menu.

  1. Uncheck the Active box by clicking on it.
  2. Scroll down and click Update Users Profile.

 

Archive a user via the user list

Select Manage People > Users.

  1. From the list of Users click the check box to the left of the required User to be archived.
  2. Click Archive.
