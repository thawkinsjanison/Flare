Search for users

The Search feature allows you to quickly find a User.  Where there are thousands of Users in the system there are search filters available to create a more powerful search.

To search for a User select Manage People > Users.

  1. Click Search for Names, Email, Username or Group Name and a search field will appear.
  2. Enter search word – either a name, email or username.
  3. Alternatively dates created before or after can be used as a search option.
  4. Or else, click in Search for Groups to activate the drop-down menu and select a group to search.
  5. User range refers to filtering around reports. Click in the box to activate a drop-down menu and select from the three options.
  6. Click Search.
  7. To perform another search click Reset Search.

The search results will appear in a Display Name column below the search box.
