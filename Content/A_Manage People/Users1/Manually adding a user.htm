Manually adding a user

Select Manage People > Users and click on Add User from the Actions drop-down menu.

  1. Complete the form details including adding a profile picture (please refer to the next section for steps on how to do this).
  2. Check the Active box to activate the User in the system.
  3. Set the User's role.
  4. Click Save User.

The details of this form can be customised to include additional information or to make certain information mandatory or hidden, see Settings > User Settings > Edit Form.
