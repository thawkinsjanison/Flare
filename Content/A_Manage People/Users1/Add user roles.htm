Add user roles

What is a role?

A Role is a collection of permissions that refer to a set of actions or responsibilities a User can perform.  A Role could be 'Basic User' or 'Tenant Administrator’.

There are the two ways to assign a Role to a User.

Add user roles – system

  1. Search for the User using the Search menu.  Or
  2. Click on the User from the Display Name List.

  1. Select Edit User from the Actions drop-down menu.

  1. Scroll down to System and select the Role and Scope of the role from the drop-down menu.
  2. Click Update User Profile.

Add user roles – extra roles

  1. Search for the User using the Search menu.  Or
  2. Click on the User from the Display Name List.

  1. Select User Extra Roles.
  2. Select the Role and Organisation entity.
  3. Click Add Role.
