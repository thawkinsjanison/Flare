Search for a Group

The Search feature allows you to quickly find a Group. You would only need it when there are a large number of Groups in the list.

To search for a Group select Manage People > Groups.

  1. Click Search for Group Name or Group Type and a search field will appear.
  2. Enter the search word.
  3. Alternatively select the date(s) the Group was created before or after.
  4. Click Search.

The search results will appear in the Display Name column.

  1. To perform another search click Reset Search
