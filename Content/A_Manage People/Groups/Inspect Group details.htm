Inspect Group details

Inspecting a Group’s details gives you access to information about the Group and its User members. The screen also allows you to add Users to the Group and to set Users with various permissions over the Group.

To Inspect a Group navigate to Manage People > Groups.

  • Locate the Group by searching for it or locating it in the list.
  • Click the Name of the Group.

You will be taken to the Group’s details screen. You will see:

  1. The Group Type of this Group; this is a clickable link.
  2. The numbers of Users: these are clickable links to lists of those Users.
  3. The Name and Identifier.
  4. You can click Administrators to add Users with various Roles over the Organisation. For example, depending on how the Roles are set up, making a User a Marker will give her/him access to Manage Learning > Marking where s/he can mark any Test Attempts that Users in this Group produce

    5. Changes using the Administrators widget also update the list in the Permissions collapsible window. You can click it to view the list.

  1. Click the Users collapsible window to view, add or delete Users from the Group. You will see the following items:

  1. Near the bottom of the window is a list of all the Users in this Group, divided into active and inactive.
    2. Click a tab to view the category.

    Each list item allows you to click the Name and be taken to the User's details screen. You can also click the User’s Org Unit and be taken to its details page.

  1. Tick the check box to the left of a User. You can then click Archive to make this User inactive in the Group (but not in the CLS). Click Remove to remove this User from the Group.
  2. In a large Group you can use the Search bar to search the list of Users in the Group by Name, email address or Username. You can also filter the list to limit it to Users who directly or indirectly report to you.
  3. To add new Users to the Group click and select User in the first drop-down list.
  4. Click and select one or more Users from the second drop-down list and click Add
