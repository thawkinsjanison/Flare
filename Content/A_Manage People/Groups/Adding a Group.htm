Adding a Group

There are two ways that Groups can be added to the CLS:

  • Manually.
  • Through an Import method.

Note that these methods create the Group(s), however neither adds Users to them. To add Users you need to use the Group details screen or import a template that lists Users’ Group memberships. Note that you could select a setting in this import to create Groups at the same time as creating User memberships. Alternatively you could use import Users to a Group and add both Users, their Group memberships and the Groups themselves by selecting the appropriate settings.

Add Group manually

Select Manage People > Groups and click on Add Group from the Actions drop-down menu.

  1. Select an Organisation.
  2. Select an Organisation Unit.
  3. Select a Group Type that the Group belongs to from the drop-down menus. These need to be set up prior to the creation of the new Group.
  4. Create a unique Name for the Group.
  5. Enter a unique Group ID.

  1. A Description of the Group can be added but is not mandatory.
  2. Check the Is Active box to activate the Group.
  3. Click the chevron to enter Administrators who have appropriate permissions over the Group.
  4. Click and select a User from the drop-down box
  5. Click and select a Role from the drop-down box and click Add.

You can add multiple Users using this process.

  1. Click Save Group.

Import Groups

If there are multiple groups to add, then this function should be used.

To access a Group import template select Manage People > Groups and click Import Groups from the Actions drop-down menu.

You can use the sample excel template or load your own.

  1. Click the template link to download the sample. Fill out the required details and save to your local machine.
  2. Click Select File to browse your machine and open the correct file.
  3. Select Upload.
