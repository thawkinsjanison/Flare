Manage Groups

To manage Groups select Manage People > Groups.

The screen shows a list of all Groups in the Tenant.

From this screen you can:

  1. Search for Groups, see Search for Group.
  2. Archive Groups (s), see Archive Group.
  3. Examine the list of Groups.

Each Group is listed, along with the Group Type it belongs to, the number of Users who belong to it and its creation date.

All but the last of these items are themselves clickable links to further information and ultimately the opportunity to examine or edit items or the members of the list.

For example clicking the number under the Users heading leads to a list of those Users with links that allow you to inspect, email or edit each User.

Note that you can reorder the list of Organisations by Name or Last Update by clicking on the appropriate column heading.
