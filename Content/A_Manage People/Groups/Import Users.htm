Import Users

To import Users into a Group, follow the steps to edit the Group and then click Import Users.

This takes you to Mange People > Users > Add or synchronise Users from spreadsheet.

The spreadsheet template has one column for Groups. Set the column name to the desired Group Type and the column value to the Group Name or Identifier. If the Group does not exist, it can be created using this value as the Group Name.

The import process could be used to add new Users and a Group or update User membership in a Groups, depending on what is specified.

The relevant settings are:

  • The Data Source drop-down allows you to specify whether Users present in the system but not in the spreadsheet should be inactivated. This does not apply to Groups, who may lose members if this is chosen but will not themselves be inactivated.
  • If Allow update of existing Users is checked then Users already in the CLS will be updated. This option could be used to add existing Users to a Group.

  • If Allow creation of new groups is checked Groups can be created during the import. A new Group will be created if the data entered in the Group column for a User does not already exist.
  • If Inactivate old group memberships is checked any Group memberships a User has that are not specified in the import file will be inactivated. This only applies to Group Types specified in the import file.

See the full details at Positions > Import Positions.
