Import memberships into a Group

To import a list of Users into Groups select Manage People > Groups and select Import Memberships.

  1. Click the template, fill it out following the headings established and save it to disk.

If a User is a member of multiple Groups then list the username multiple times, one for each Group membership.

Note that you can also use this template to deactivate memberships using the Is Active column.

  1. Click the checkbox to specify that any Group memberships a User already has that are not specified in the import file will be inactivated. This only applies to Group Types specified in the import file.
  2. Click the checkbox to allow the creation of any Groups specified in the spreadsheet but not found in the CLS.
  3. Enter an email address where the results of the import will be sent.
  4. Select your updated template.
  5. Click Import and email result.
