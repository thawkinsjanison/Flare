Edit Group

Use this function when Group details need to be changed. Follow the steps to inspect a Group’s details and click Edit Group.

You can edit the fields that were already described in Add a Group and click Save Group.

You can also:

  1. Click Delete Group to delete the Group.
  2. Click Admin Delete to delete the Group and the Users who are members of it. This is a very serious action to take so a pop up box will appear showing exactly what will be deleted if you click Confirm. You will need to first remove these Users from any other Groups and relationships they have. You will also need another User with an administrator account to ‘co-sign’ the action.
