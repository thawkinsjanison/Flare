Importing Catalogue Item Tags

To Import Catalogue Item Tags you will need to know the Identifier of each Catalogue Item you would like to apply a Tag onto. An easy way to gather this information is with a Customisable Report using the Catalogue Item Entity and making sure the Name and Identifier columns are selected. You should also incorporate filters to focus your list of catalogue items.

Read the Customisable Report guide for more details on this feature.

Browse to Manage Learning > Manage Catalogue.

  1. Select Import Catalogue Item Tags.

  1. Select Template. A template file for Catalogue Item Tag Imports will download.

  1. Open the file and enter your desired Catalogue Item Identifiers onto the Catalogue Item ID column.
  2. Enter the Tag Type to use.
  3. Enter the Tag which will apply to this catalogue item.
  4. Enter True in the Is Active column.

Note: There are also two columns for Custom Attributes. These do not apply to the standard Tag import and can be removed.

  1. Save the template somewhere where you can find it easily again.

Back in Social Learning.

  1. Select File… or Drag and Drop your saved file into the Spreadsheet File area.

  1. Optionally, select or clear Allow Tag Type Creation.
  2. Optionally, select or clear Allow Tag Creation.

Note: These options allow the import process to create tags and tag types from the report if they do not already exist within the system. In this example, we are going to clear them as we are using tags and tag types which already exist within the system and do not want to risk creating tags as a result of spelling mistakes entered into the spreadsheet.

  1. Enter the Email Address where the import summary will be sent.
  2. Select Import and email results.
