Enabling Tag Types on Items

Browse to Settings > Social Learning.

  1. Select the icon in the Tags Section.

  1. Select Tag Type.

The Tag Type will be added to a list.

  1. Optionally, enter an Alternate Name for the Tag Type. Alternate Name will override the Tag Type name when displayed on a catalogue item.
  2. Optionally, select Filterable. Selecting Filterable allows users to filter search results to this Tag Type.
  3. Select the icon to remove it from this list.
  4. Select Save.
