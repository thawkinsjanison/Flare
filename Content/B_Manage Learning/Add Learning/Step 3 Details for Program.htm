Step 3 Details for Program

A Program is a collection of Learning Assets that can be delivered to Users. It is very similar to a Catalogue Item. The difference is that it contains more than one Learning Asset, see Programs for more information.

In this step you are adding its Components, i.e. its linked Learning Assets and optionally setting a Program Template.

  1. You do not have to select a Template to determine how the learner sees this Program. You can if you choose click in this box and select one. See Program Templates for more information.
  2. In the Components sections you select the linked Learning Assets in the Program and specify other settings that relate to them, such as Prerequisites. See the discussion of Components in Add Catalogue Item for full details.
  1. Click Next
