Step 2 Description

In the Description screen you enter details that will be used to create the Catalogue Item that your Learning Asset is linked to. See Add Catalogue Item if you need further information about the details.

To complete this screen, select the window headed What are the details of the….? [this sentence will end differently depending on the Learning Asset you are adding, e.g. it may read: What are the details of the Learning Event?]

  1. Enter a Name and Identifier.
  2. Add a short summary in the Overview.
  3. Click Add Learning Objectives to open a window where you can add the Learning Objectives.
  4. Enter the Duration in minutes.
  1. If you check Has Certificate extra fields pop up and you can enter a Certificate Code. The certificate template already specified in Settings (see Certificates) is displayed; alternatively you can select another template using Select files.
  2. Click the chevron next to What tags would describe this ...? and enter Tags if desired. Tags are broad terms that describe the topic of a Learning Asset (see Manage Tags). In Description Tags you can select pre-existing tags and in New Tags you can enter new tags.

  1. Click the chevron next to Who is the …? click in the box and select the manager’s name from the options presented in the drop-down list.

    2. Note that this will be labelled differently depending on the type of Learning Asset. If you are creating a Test, you will be asked to select its Marker.

  1. If desired, click the chevron next to Advanced Details. Select an Archive Date and enter Version Information, a Provider, an Accreditation Number and a Price if desired.

  1. Upload Large and Small Thumbnails if desired. The size of small thumbnails should be 47 x 22 pixels; the size of large thumbnails should be 292 x 70pixels.

  1. Click the chevron next to Point Allocation. These relate to Continuing Professional Develoment (CPD). See CPD and Catalogue Items for details of how to set this up.
  2. Click Next.

You have now set up the details of the Catalogue Item for your Learning Asset. Clicking Next takes you to Step 3 where you provide details for the Learning Asset you are adding. For help go to Step 3 for the type of learning you have chosen.
