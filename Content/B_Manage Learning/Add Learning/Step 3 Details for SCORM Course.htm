Step 3 Details for SCORM Course

SCORM stands for: Shareable Content Object Reference Model. SCORM courses have been created in 3rd party applications such as Lectora and Captivate. Full details are at Add Learning Asset - SCORM Course.

Note: prior to adding this type of learning please ensure you have a SCORM compliant zip file. The default upload limit is 200MB. If you require a larger limit please contact the Janison Development team with a request to increase this size.
