Add Learning

Add Learning is a wizard, i.e. a setup assistant that offers you a series of dialog boxes to complete the steps required to create an enrollable Learning Asset in the CLS.

The wizard creates the Learning Asset and also puts it into a new Catalogue Item with the same name, so that it is possible for Users to enrol in it.

Note that as an alternative to using the wizard, there are manual methods to create Learning Assets, e.g. through Manage Learning > Manage Learning Assets > Add Learning Asset. The manual methods do not link to a Catalogue Item with the same name, as happens using the wizard.

There are a many types of learning in the CLS, see Types of Learning Asset.

A number of these types of learning can be added to the CLS system via the wizard. They are:

  • Learning Event: made up of one or more Sessions where a user must attend a venue
  • Other Learning: external learning already undertaken by a User
  • Program: a Catalogue Item containing more than one Learning Asset
  • SCORM Course: a course created in a 3rd party application such as Lectora and Captivate
  • Trackable Document - a document used to deliver learning material in PDF, Audio, or Video formats

If it has been enabled in the Test Designer settings, you can also create a:

  • Test: a container for Questions

    • Note that the wizard simplifies the process to make creating an enrollable Test easy. The default Discipline is selected and a new Assessment Module is also created (in addition to the new Catalogue Item that the wizard creates for every type of Learning Asset).

    Note also that:

    • question banking is disabled (you cannot reuse Questions across tests).
    • you cannot search for pre-existing Questions to add to a Test.
    • when you unlink a Question from a Test this deletes the Question, not just the link.

Depending on what has been configured by Janison in your settings for Learning Assets, the wizard may also create:

  • Assessment Events: Assessment at a specific time and place
  • Lti Events: learning at a completely separate website that the CLS links to

The Add Learning wizard can be accessed from two locations.

  • Select Manage Learning > Manage Catalogue > Add Learning Wizard, or

  • Select Manage Learning > Add Learning

Note that if you discontinue the wizard before you complete all the steps, a Catalogue Item and Learning Asset are both created. You can locate them through Manage Learning > Manage Catalogue. You will find the Catalogue Item listed amongst the items with Draft status. After selecting it you can edit both the item and the asset.
