Step 3 Details for Test

Tests in the CLS are made up of Questions. Tests also include settings about how the Test is to be timed, marked, randomised, ordered and delivered. Further information about assessment Tests is found at Assessment Tests.

Tests can play a number of roles in the CLS. They can be part of learning activities and are very important in assessment activities. The Add Learning wizard creates assessment Tests.

The wizard simplifies the process and makes it easy to create an enrollable Assessment Test. It selects the default Discipline and creates a new Assessment Module, in addition to the new Catalogue Item that it creates for every type of Learning Asset. Note also that:

  • question banking is disabled (you cannot reuse Questions across Tests).
  • you cannot search for pre-existing Questions to add to the Test.
  • when you unlink a Question from the Test, this deletes the Question, not just the link.

In this screen you enter all the information about the Test and its linked Questions.

  1. Click Preview Test to see how the Test you are creating on this screen will look in the Test Player. You can do this at any time while you are at this screen.
  2. Specify how the Test will be marked from the Marking Options drop-down menu. Most question types in the CLS can be automatically marked. The type of marking is listed for each question at Assessment Question Types

Pass / Fail and Resit Settings

Select the Pass /Fail Resit Settings window to open it.

  1. You must enter a Pass-mark, i.e. the percentage of correct marks required to pass the Test. If you enter 0, Test Attempts will be given the status of Completed and a pass/fail status will not be recorded.
  2. Enter the number of times a student can attempt this Test in number of attempts allowed.
  3. Click the checkbox if a student re-sitting the Test is only presented with Questions that were answered incorrectly in an earlier attempt.

  1. Select the type of feedback students receive (if any) from the drop-down menu in the Feedback box.

Timing Options

Click the chevron to open the Timing Options window.

  1. Enter the Duration: the number of minutes available to the User for sitting the Test.

Questions

Click the chevron to open the Questions window.

  1. Click Add question to add a new Question. This takes you to the Add Test Question screen where you can create the Question, see Assessment Question Types for the required details. Save the Question.
  2. Once you have saved the Question it appears on the list of Questions in this Test.
  3. You can enter the Question’s relative Weight.
  4. You can click the check box to indicate that it is mandatory. This is relevant if you are offering students a random selection of Questions from this Test (see 13).

  1. Under the Actions heading you can find icons to view (eye) the Question, edit it (pencil), delete it (minus) and reposition it (cross-hair) in the list. Note that in the wizard, when you delete a Question from the Test this deletes the Question, not just the link to it.

Randomisation and optional Questions

Click the chevron to open the Randomisation window.

  1. Select a Randomisation option if desired. You can randomise the order that Questions are presented to students. Alternatively, you can randomly select Questions from those attached to the Test. If you choose this option you will be asked to enter the number of Questions that should be selected from the total number. Note that this option will also randomise the order of the selected questions.

Player Skin Settings

Click the chevron to open the Player Skin Settings window.

  1. Click and select a skin for the Test Player if these have been defined in the CLS.

Sequence

Click the chevron to open the Sequence window.

You will see a number of standard text boxes where you can enter text and other resources that will be presented to students at the beginning and end of a Test Attempt.
