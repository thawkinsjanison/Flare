Certificates – Introduction

If Certificates have been enabled for a Catalogue Item they are generated whenever a User successfully completes it.

A certificate is based on a PDF template. The CLS populates this PDF template with the appropriate data into the matching fields in the template. Thus, the template can be customised for each User, for example with the addition of the User’s name and their completion date.

When you set up Certificates you can create a generic Certificate template for all your Learning Assets. Alternatively, you can create specific templates for particular types of Learning Assets.
