Access and View Certificates

There are multiple places in the system where Learners can access their certificates.

My Learning Transcript

Certificates can be accessed from My Learning Transcript as highlighted below. See the document on My Learning Transcript for further information on how to enable the Certificate column.

Catalogue Item

When the learner launches a Catalogue Item an interim screen is encountered where the certificate can be accessed as shown below. The page may look different depending on the Catalogue Item Template selected. See the document on Templates for further information. In the below example the Default Vertical template has been used.

My Learning

Learners can access their certificates from the Catalogue Item Tile on their My Learning page. The certificate icon links to download the certificate.
