Creating and Editing a Certificate Template

Before you start creating Certificates you need to ensure that you have access to software capable of editing PDF documents, e.g. Adobe Acrobat Pro.

Use MS Word (or equivalent) to generate the certificate layout, graphics and all static (non-changing) text and save it as a PDF file. You edit this file in Adobe Acrobat or Distiller and insert the fields that the CLS will later populate with live data (e.g. names).

Create a Certificate Template

You will find an example of a word template here.

  • Open Microsoft Word (or equivalent)
  • Create a document and add the layout elements and static text as desired
  • Leave blank areas covering the whole width of the page where each of the fields will be placed (e.g. the User name)
  • Save the document as a PDF file (use Save As)

You will now need to insert fields in this PDF document using Adobe Acrobat Pro or similar. A list of the fields that you can include is below, see Field names.

You can find a sample of a certificate with Form fields added here If you open it in Adobe Acrobat Pro the fields will be visible and editable. The form fields added to this certificate template are: FullName, NameOfCourse, DateOfCompletion and CertificateNumber.

This is a sample of the above certificate once it has been generated with the form fields populated.

Open the PDF document in Adobe Acrobat Pro

  • To add fields, select the Forms tab followed by the Add or Edit Fields option (alternatively, press Shift+Ctrl+7)

  • Select the Add New Field button followed by the Text Field option from the drop-down menu

  • Place and size a box on your PDF document using the selected ‘text field’ device. Make sure that you leave ample space in the field for long names etc.

  • Insert the appropriate fieldname (in this case ‘FullName’) and press the enter key

Edit Template Fields in a Certificate Template:

To edit fields in the template you need to open it in Adobe Acrobat, or a similar PDF writer and have the Add or Edit Fields option on.

  • Double click on the Field box to open the ‘Properties’ for that field

  • Select the Options tab to insert a default value if desired
  • Use the Appearance tab to set the font and font size
  • The General tab can be used for changing the text field name (if for example, a spelling mistake was made)
  • Save the new certificate

Notes:

  • Format the fields with the same typeface, font size, etc as the rest of the document (in Appearance, above)
  • Make sure that the text field name is exactly the same as the matching CLS field name, see Field names.
  • Set most fields to be the full width of the page and then centre align them to allow for variations in size (e.g. the length of different names).
