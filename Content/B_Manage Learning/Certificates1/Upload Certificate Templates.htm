Upload Certificate Templates in Settings

One template can be set as the Default template which will apply to all the Learning Types in the system. Alternatively you can set specific Certificate Templates for specific Learning Asset types.

To upload a Certificate Template:

  • Navigate to Settings > Tenant Settings.
  • Click the chevron to expand the Certificate Templates section


To set a specific Certificate Number Format enter it in the ‘Format for Certificate Number’ field, see Certificate Number

If you are only using one Certificate Template for all Learning Types upload a certificate to the Default Certificate Template.

If you are using unique Certificate Templates for different Learning Types upload a separate certificate to each Learning Type.
