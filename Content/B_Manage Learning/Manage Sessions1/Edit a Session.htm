Edit a Session

You may need to edit a Session after you have created it, for example, to change its Status to ‘Cancelled’. And if you need to modify the marks in a completed Session, you first need to change its status to 'Published'.

To edit a Session, select Manage Learning >Manage Sessions. Click Manage next to the Session that you want to edit

This takes you to the Manage Sessions screen for that Session. Click Edit Session from the Actions drop-down menu.

This takes you to the Edit screen for that Session where you can edit all fields and save the Session. See Add a Session for details of these fields.
