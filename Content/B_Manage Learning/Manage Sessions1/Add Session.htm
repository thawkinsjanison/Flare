Add Session

Once a Learning Event has been created you can add Sessions to it.

Note that you can also create Sessions through Manage Learning> Manage Learning Assets or when you create a new Learning Event with the setup assistant at Manage Learning> Add Learning.

To add a Session in Manage Sessions select Manage Learning >Manage Sessions.

Click the Add Session button.
