Learning Event Settings

There are a number of setting that relate to Sessions and Learning Events, including.

  1. Group Type – click and select a Group type
  2. RPL can be added to the options that a Session marker can choose from. It is then compulsory to add some notes
  3. These option enables the movement of whole groups and org units between waiting lists and attendee lists
  4. Click to show outstanding Sessions on learner User’s calendar
