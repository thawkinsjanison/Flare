Conditional Release

You can use Conditional Release to set time or date based periods during which a program component is active for use. There are two ways to set Conditional Release.

  • You can set a time for when a component will be made active within a program enrolment.
  • You can set a time for when a component will be made inactive within a program enrolment.

Learners can see the details of both but cannot access inactive content.

Conditional Release also works with Prerequisites. An asset does not become available to a learner unless both the prerequisites are met and the asset is available for use under Conditional Release rules.
