Manage a User’s Conditional Release

A situation might arise where an administrator needs to modify a learner’s enrolment. On a per user basis, an administrator can change the date on which an asset is released or the date an asset becomes locked.

Browse to Manage Learning > Enrolments.

  1. Select View More () to go to a learner’s enrolment.
  1. Select the Program Components section.
  2. Select the icon for either the Release Date or the Lock Date to edit them. A dialogue box will appear.
  1. Enter a new release or lock date.
  2. Select Save. A dialogue box will appear.
  1. Select Yes to make this change to all learner enrolments for this program which share the same enrolment date.

or

  1. Select No to only change the current learner’s enrolment.
