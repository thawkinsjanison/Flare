Schedule a Report

You can schedule a Report to run in the future, including running on a regular basis. Its results can be either saved to disk or emailed to a recipient.

If you have specified in the Report that the export results use an Excel Template then the template will be applied to the results before they are exported to an Excel file.

To schedule a Report navigate to Manage Learning > Customisable Reports and locate the required Report in the list. There are two scheduling options:

  1. Schedule the Report to be run and emailed.

    2. Click the @ icon. This takes you to Schedule a Job in Settings where you can specify when the Report should be run and who is emailed the file containing the Report results. This is the recommended scheduling option.

    If you want the Report emailed to more than one address, insert a semicolon between the addresses.

  2. Schedule a Report to be run and saved to a File.

    3. Note: This is a specialised feature and you will not have access to the file you have scheduled. For this reason we would recommend scheduling to email.

    Selecting the icon takes you to Schedule a Job in Settings where you can specify the scheduling settings.
