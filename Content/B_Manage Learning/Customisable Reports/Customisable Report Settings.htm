Customisable Report Settings

There are several settings for Customisable Reports. Navigate to Manage Learning > Customisable Reports and click Settings.

  1. Check this setting if you want to be able to apply templates to exported results, see Report Templates.
  2. Check this setting to enable the private Scope for a saved Report. This limits it to the User who created it.

  1. Check this setting to enable Scopes for a saved Report: Me and Others, My Organisation and My Organisation Unit.
  2. Click to open the Enabled entity types collapsible window. Check all the CLS entity types that can be reported on.
  3. Click Save settings.
