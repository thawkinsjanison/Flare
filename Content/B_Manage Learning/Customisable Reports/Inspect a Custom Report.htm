Manage a Custom Report

To locate a Custom Report that has previously been created and saved, select Manage Learning > Customisable Reports.

Locate the Custom Report in the list or search for its name using the Search menu.

When you click the name of the Report, it runs and the subsequent screen shows its results.

The top of the results screen offers three options:

  1. Click Refine Report to inspect and edit the Report instructions (e.g. modify the details selected or change the filters). Make any required changes, see Create a Custom Report for details.
  2. Click Save Report to save the Report. If you have not previously made any changes this is not necessary
  3. Click Export to export the results of your report to your computer’s Downloads folder. The default format is an Excel file (.xlsx). If you click the box to the left you can export to other formats: Comma-separated vales (.csv) and an Acrobat reader document (.pdf)

After you have located a Report in the Customisable Reports screen you can also:

  1. Edit the Report instructions.

    2. Click the pencil icon. This takes you to a screen where you can change the components of the report. See Create a Custom Report for details.

  1. Schedule the Report to be run and emailed.

    2. Clicking the @ icon takes you to Schedule a Job in Settings where you can specify when the report should be run and who is emailed the result. Note that if you want the report emailed to more than one address, insert a semicolon between the addresses.

  2. Schedule a Report to be run and saved to a File.

    3. Clicking the @ icon takes you to Schedule a Job in Settings where you can specify when the report should be run.
