Export the Report Results

Once you have run a Report, its results screen has an option at the top that allows you to Export the results to a file. Depending on your browser settings, it will be saved to your computer’s Downloads folder.

The results can be exported in the following formats:

  • PDF
  • Microsoft Excel
  • CSV (comma separated values)

Click and select the required format from the drop-down list and click Export.

If you have specified that export results from the Report use an Excel Template then the template will be applied to the results. Only the Excel format is available when using a template.
