Customisable Reports - Introduction

Customisable Reports provide the ability to create and run detailed reports on key entities in the CLS system. A Report is essentially a set of instructions describing:

  • a particular component (entity) of the CLS, and
  • the information in that entity to be displayed.

The process involves several steps, some of which are optional:

To create and run a Report:

  • Select the entity you wish to report on
    e.g. Users.
  • Optionally apply filter(s) to precisely define part of that entity
    e.g. inactive Users with the role of Basic User.
  • Select the entity information to report on
    e.g. User’s names and email addresses.
    Optionally order the way this is displayed.
  • Run the Report and view its results.
  • Optionally export (save) the results in various formats
    e.g. as an Excel file; optionally using a template to display these results in Excel.
  • Optionally name the Report and save it (i.e. the instructions) for use in the future.

There is also an option such that when a Report is run it pauses to allow a filter value to be entered on the spot (see Building Filters).

Note that some of the steps listed above are optional. The only essential steps are:

  • Select entity,
  • Select entity information,
  • Run the Report.

When you run the Report its results are shown on the screen and these can be exported.

The Report itself can be named and saved so that it can be run (and possibly edited) later on.

The key features of Customisable Reports are:

  • sophisticated filters can be applied to the entity (including Custom Attributes with limitations).
  • the details reported can be selected (including Custom Attributes).
  • Report results can be exported to Excel format.
  • Report can be saved and modified for re-use at a later date.
  • Reports can be scheduled to run automatically.
  • Excel Templates can be used to order and visualise the data reported.
  • security.
  • in-grid view of data.

Note that not many Users are authorised to use the Customisable Reports tool, so they would not see this menu item. Thus reporting is "wired" into Janison’s Security framework which ensures that Users cannot see information that they’re not authorised to see.

To access Customisable Reports navigate to Manage Learning > Customisable Reports. You will see a list of previously saved Reports.

Running a Report

You can either create a new Report and run it straight away or locate a previously saved Report and run that.

To run a previously saved Report, navigate to Manage Learning > Customisable Reports and locate the Report.

Clicking its name runs the Report and displays its results on the screen.

To automatically run a previously saved Report, schedule it to run.
