Using Custom Report Templates

You can use an Excel template to specify and order the fields in the entity that you want to report on. You can also create useful visual displays of that data using Excel tools such as charts.

Refer to Excel documentation for details of how to build these visual display tools in an Excel file. The example below is based on the sample template available in the CLS, which uses a chart to build a pie chart displaying the proportion of genders in a list of Users.

Select Manage Learning > Customisable Reports.

  1. Click Settings

  1. Click the checkbox next to Enable Excel Template based reports and click Save settings

  1. As a result, creation or editing a Report has a Template option. Click to download the sample template.

  1. You can customise and save this template as desired.

    2. The DataTemplate page specifies the fields in the entity that will be reported. Note that the names under PropertyName must match the names of the fields in the CLS entity.

  1. Select your template file and apply to the Report. You can detach the template by clicking the cross icon.

Note that you can still filter the entity in this Report but the fields that will be reported are determined by what was specified in the template. You cannot change the onscreen field checkboxes as they will mirror the template.

  1. Run the Report by clicking the icon on the bottom of the page

  1. Export the results as usual. Only the Excel format is available when using a template.

This will export the results to an Excel file with the same name and format as your template file.

  1. Open the file in Excel. The RawData page will show the data generated by the report.

You will need to click Enable Editing so that the pie chart on the Chart page is updated with the new data.
