Create and Run a Custom Report

After you have created a Report, you run it and display its results. The results screen will offer you the options to export these results, save the Report itself or edit the Report.

There are two ways to create a new Report: you can create one from scratch (see below) or you can locate and run an existing Report and then refine (edit) it, see Inspect a Custom Report.

To create a new Report select Manage Learning > Customisable Reports and click Create a Report

  1. Click under Entity and select the required entity from the drop-down list. You can only select one entity per Report.

    2. In this example Enrolment has been selected.

  2. If desired, select parts of the entity by building Filters to apply to it. See instructions in Building Filters.

    3. In this example, a simple filter selects enrolments where the User has a Basic User Role.

If you have enabled the setting, you can also specify that any export of the results of this Report uses an Excel Template. In this case, the only export format will be Excel.

  1. In Select Columns check the details (fields) of the entity that you want included in your Report.

    2. The fields that you can select are different for each entity. Note that these options also give you a useful sense of the information that is stored in the CLS for an entity.

    In this example 3 fields have been selected. To simplicity the screenshot, unselected fields have been removed from the image.

  1. Click Set column order if you want to customise the display order of the columns of information.

    2. Change the order by dragging rows (e.g. Full Name in the example below) up or down in the list and click OK

  1. Click Run Report to display the results of the Report.
