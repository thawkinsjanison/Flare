Save the Report

Once you have run a Report, its results screen has an option at the top that allows you to Save the Report.

In the Save menu enter a name and an optional description.

You can limit the scope of this Report by selecting from the Scope drop-down list. This restricts who can see and run the saved Report. The Scope options are set in the Customisable Reports settings.

If you have reached the Save screen through refining (editing) an exisiting Report, you can save the Report as a copy (rather than overwriting the existing one) by checking the Save as Copy checkbox underneath the Scope option.
