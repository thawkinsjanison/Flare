Managing and Inspecting Tags

The CLS allows administrators to inspect Tags and check the items they are attached to. To inspect Tags select Manage Learning > Manage Tags.

  1. The Tags screen is divided into Active and Inactive (Archived) Tags. Click the tab to see the members of that kind of Tag.
  2. Individual Tags can be located in the list or by clicking Search for Tags to search within it.
  3. The list can be ordered by Name or Update date by clicking on the heading. A black triangle indicates which of these currently determines the ordering.
  4. The information for each Tag includes the numbers of items in each relevant category in the CLS that have this Tag attached.

Each one is also a link to a list of the members of this category of items, thus making it possible to ultimately inspect and/or edit details of and enrolments in the members of this category.

Inspect a Tag's details

To inspect a Tag select Manage Learning > Manage Tags and click the name of the Tag.

  1. The name of the Tag is shown, along with its description. If it has a Tag Type this will be displayed here; in the example above it does not.
  2. The numbers of items in each relevant category in the CLS that have this Tag attached are shown.
  3. Click Edit Tag to edit the Tag
