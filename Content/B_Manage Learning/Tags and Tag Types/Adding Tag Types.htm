Adding Tag Types

To add a Tag Type select Manage Learning > Manage Tags > Manage Tag Types and then click Add Tag Type.

  1. Enter a name for the new Tag Type
  2. Enter a description if required
  3. If desired click Select File to upload a thumbnail image of 64 x 4 pixels. Note that this is only seen when the Tag Type is inspected
  4. Click the triangle to the left of Applicable Documents to select the kinds of documents where this Tag Type can operate. If nothing is selected then all the documents are considered applicable.
  5. You can enter a Tag that is a member of this Tag Type by clicking and typing in the Enter a new tag box and then clicking the plus icon to complete the process.

After this you can click the minus icon to delete it or click the pencil to edit its name save the change by clicking the tick icon.

Note that you can enter existing and new Tags in this list. New Tags will be created after the Tag Type is saved.
