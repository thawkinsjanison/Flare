Adding Tags

To add a Tag select Manage Learning > Manage Tags and then click Add Tag.

To create a new Tag:

  1. Enter the Tag Name
  2. Click and select an existing Tag Type if desired
  3. Enter a number in Sort Order. This number will be compared in relations to adjacent items.
  4. Enter a Description
  5. Select an image to set as a thumbnail if desired.
  6. The Tag is active by default. Check the Inactive box to reverse this if desired.
  7. Click Save Tag.
