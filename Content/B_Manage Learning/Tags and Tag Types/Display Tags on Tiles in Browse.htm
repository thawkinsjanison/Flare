Display Tags on Tiles in Browse Learning

Tags can be displayed to Users on Catalogue Item tiles when they are self-enrolling from Browse Learning. For this to be possible the Tags must be linked to a Tag Type.

  • To enable the tags to appear on the tiles Browse to Settings > Browse Learning Settings and open the Catalogue Appearance section.
  • Under Side Filters select a Tag Type and an option to Show on Tile will be available.
  • Check the box next to the Tag Types you would like to appear on your tiles.
