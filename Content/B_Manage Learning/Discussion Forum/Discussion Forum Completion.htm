Discussion Forum - Completion Settings

Requirements for a participant to Complete can be set in the Completion Settings section.  If the Learners must post to complete option is checked a field to set the number of posts required can be populated with the required number of posts the participant must make to register a status of Complete.

Progress status can be viewed in a number of places including the Learner’s My Learning Transcript and the Forum tile.  This information can also be viewed in Custom Reports.
