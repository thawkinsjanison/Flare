Discussion Settings

Discussion settings can be configured at a site level which will act as the default when creating a new discussion. Most of these settings are also available at a Discussion Forum Asset level too which if set, will override the site setting.

Browse to Settings > Discussion Settings.

  1. The Rating Type controls if users are allowed to rate other posts. None will remove the ability to set a rating. Like or Dislike will add an interactive rating feature to each post.

  1. Post Types control whether post types will appear on the forum.

Checking a Post Type will make the corresponding type appear:

  • For selection when making a new post.
  • In My Posts. Posts made by all users enrolled in the discussion are filtered into their category and accessed here.

  1. Allow posts to be deleted will add an option for administrators to delete posts.

  1. Allow posts to be edited will add an option for administrators to edit posts.

  1. If an email address is added to this field, it will become the recipient of reported inappropriate behaviour.
