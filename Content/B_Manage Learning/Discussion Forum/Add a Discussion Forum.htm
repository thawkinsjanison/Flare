Add a Discussion Forum

  • To add a new Discussion Forum navigate to Manage Learning > Manage Learning Assets.

  • Click Add Learning Asset.

  • Select Discussion Forum from the list.

  • Create a name and identifier for the Discussion Forum.
  • A description about the forum can be added but is not mandatory.
  • Set the forum to Active.

A Discussion Forum is a type of Learning asset and needs to be put into a Catalogue Item for participants to access them.  To learn more about the steps to do this please read the documents on Add catalogue item and Enrolments
