Note how the original and discounted prices are displayed in the above screenshot.Subscriptions – Updating benefits

Once users are enrolled in a subscription, if you change the subscription a popup will be displayed with an option to update existing subscribers to the new benefit.

If you selected No, a button to update all subscribers will become available to push the subscription changes out to existing subscribers at any time.

If you do not Update All Subscribers, you could have different subscribers on different versions of the same subscription.
