Subscriptions - Initial settings

Browse Learning

Browse to Settings > Browse Learning Settings. The following settings need to be selected:

  1. Price.
  2. Full Catalogue.
  3. Subscriptions.
  4. Recommend Browse Learning Search is on as well.

  1. Select Save Settings.

Self-Enrolment

Browse to Settings > Self-enrolment Settings

  1. Select Self-enrolment (Paid) for the Default Enrolment Method.
  2. Optionally select Allow Enrolment Method override per Catalogue Item and specify the options relevant to your organisation.

Payment Settings

Enable a payment gateway for your users to be able to pay for the learning. Refer to Settings > Payment Settings for further information on this feature.

Enable Tag Type

Browse to Settings > Points Settings.

Check Enable Tag Type.
