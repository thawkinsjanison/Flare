Create tags for subscriptions

Browse to Design > Manage Tags > Manage Tag Types > Add Tag Type

  1. Enter a Name and Description.
  2. Add the tags required for your subscriptions. As a minimum you only need one tag to apply a range of discounts. In this document we use ‘Discounted’. Other examples include: 10%, 20%, 33%, 50% 80%, 100% or Bronze, Silver, Gold & Platinum.
  3. Select Save Tag Type.

For more information on Tags and Tag Types refer to Learning > Tags and tag types.
