Add Subscription to catalogue

Like all Learning Assets, a subscription needs to be added to a Catalogue item for users to be able to access them.

Refer to the document on Catalogue Items for further information.

The easiest way to do this is to select Add it now at the top of the subscription.

Tag catalogue items

For Catalogue Items to be included in a subscription, they need to be tagged with the tags listed in the Benefits section of the Subscription.

  1. Select on the Tags section of the catalogue item.

  1. Select the appropriate tag created earlier.
  2. Select Done.

Note: Make sure the catalogue item has the following set.

  • Price

  • Enrolment Method set to Self-enrolment (Paid)

  • A Payment method needs to be set, for example
