Subscriptions – Create subscriptions

Browse to Manage Learning > Manage Learning Assets.

  1. Select Add Learning Asset.

  1. Select New Subscription.

  1. Enter a Name.
  2. Enter a Description.

Note that the description will appear on the subscription tile under Check out the details.

  1. Select Save Subscription.

Benefits

Benefits are the discounts applied to Catalogue Items when a user enrols.

  1. Select the icon on the Benefits section.

  1. Enter required Discount Rate.
  2. Select the required For Tags. We are using “Discounted” as one of the tags from the subscription tag type.
  3. Select Save.

You can add several benefit groups, each with a different discount rate. The Catalogue items tagged with the corresponding tags will have the relevant discounts applied to them.

Note: You can have positive or negative benefits but not both in one subscription. You can create multiple subscriptions and have a negative and a positive subscription apply to the same catalogue item. Negative subscriptions are not seen by the end user, it will just affect the price displayed to them.

Subscription Period

  1. Select on Subscription Period section.

Note: If no subscription period is specified, then the subscription will run indefinitely.

To set a subscription to run for a fixed amount of time.

  1. Enter the required Subscription Period.
  2. Select the required Subscription Period. This is Days, Weeks or Months.
  3. Select Save.

To set a subscription that ends on a specified date.

  1. Select Expires on the selected Day/Month following Subscription.
  2. Enter Date.
  3. Select Save.
