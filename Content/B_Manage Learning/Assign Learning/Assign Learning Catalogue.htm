Assign Learning – Catalogue Item

Learning can be assigned to Users using various methods, see Assigning learning - introduction.

This method creates a link between a Catalogue Item and Enrolled Users.

  • Select Manage Learning from the main menu
  • Select Manage Catalogue (left menu)
  • Click the number of enrolments for the Catalogue Item you want to enrol Users in

  • This takes you to the Enrolments section of the CLS with that Catalogue Item and a list of all its existing enrolments. Click Enrol

  • Select User in the top drop-down list, then search for the User you want to add in the second drop-down

  • Select the User and click Add
  • The User is now enrolled in this Catalogue Item.
  • You can go back and select another User in the top drop-down list and hence add multiple Users. Click outside the box to close it.
  • Note that instead of a User, you can also select Organisations, Organisation Units or Groups and then choose from the second drop-down list appropriately. This makes it possible to enrol more than one person at a time.
