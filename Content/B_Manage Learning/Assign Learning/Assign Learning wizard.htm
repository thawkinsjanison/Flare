Assign Learning wizard

Assign learning is a wizard, i.e. a setup assistant, that offers authorised Users a series of dialog boxes to complete the steps required to enrol learner Users in Catalogue Item(s) and schedule the enrolment process (e.g. enrol now or in the future). You can select multiple items for Users to enrol in and select multiple Users and groups to be enrolled.

Note that the learning material must have been created prior to running the wizard.

To assign learning via this wizard select Manage Learning > Assign Learning.
