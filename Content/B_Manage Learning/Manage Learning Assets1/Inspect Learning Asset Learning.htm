Inspect Learning Asset - Learning Event

A Learning Event is training that takes place when a learner User actually attends a venue, including a virtual venue (webinar). One event is made up of one or more Sessions which learner Users attend. Sessions are managed in Manage Learning > Manage Sessions, see Manage Sessions for details about Learning Events and Session management.

To inspect a Learning Event select Manage Learning > Manage Learning Assets and then click the name of the Learning Event you want to inspect. You will be taken to the inspection screen for this event. Full details of the inspection screen are found with the rest of the information about Events and Sessions here.
