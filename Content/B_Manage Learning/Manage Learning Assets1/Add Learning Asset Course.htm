Add Learning Asset - Course

A Janison Course contains learning content and can also contain Tests which a learner must undertake to complete the course.

A Course is the container for one or more Modules. The Course does not directly contain any content or pages: the content exists in the Modules themselves via a series of Pages.

To add a new Course select Manage Learning > Manage Learning Assets and then click Add Learning Asset.

Select New Course from the options offered.

You will be taken to Design > Courses > New Course. See Courses for details.
