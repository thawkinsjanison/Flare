Add Learning Asset – Plan

A Learning Plan is developed by an individual after an analysis of the knowledge and skills s/he needs to achieve career and personal goals. The plan schedules particular learning that s/he would like to do in order to achieve these goals. This is a powerful type of Learning Asset and you will find a full description in Manage Learning > Learning Plans

To add a Plan select Manage Learning > Manage Learning Assets and then click Add Learning Asset. Select New Plan from the options offered. To continue follow the instructions in the document above.
