Add Learning Asset -Trackable Document

Trackable documents are used to deliver learning material in the following formats:

  • Document e.g. a PDF file or an online website
  • Audio
  • Video

Learner Users enrol in a Catalogue Item that contains the Trackable Document Learning Asset. Users are offered the choice of viewing or downloading the document.

A document that is a website is launched automatically in a separate browser window when the User clicks the document name.

User progress through the Trackable Document is monitored by the CLS and classified as Not Attempted, Incomplete or Complete as appropriate.

To add a Trackable Document select Manage Learning > Manage Learning Assets and then click Add Learning Asset. Select New Trackable Document from the options offered.

  1. Enter the Name and Identifier
  2. Enter Description
  3. Select the Type of document from the drop-down list
  4. Enter the Page Count if applicable
  5. You can add a verification step where the learner must access the document then click a checkbox declaring that they have read and understood it. To do so, click and select Verification Checkbox
  6. Enter the Document Author and Provider if desired.
  7. If the document is a website enter its online address in Document URL.
  8. Browse and select a file to upload by clicking Select a file. The tick icon indicates that your file has been selected. You can unselect it by clicking the cross icon
  9. Click Save Trackable Document
