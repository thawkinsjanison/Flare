Add Learning Asset - Learning Event

A Learning Event is training that takes place when a learner actually attends a venue, including a virtual venue (webinar). It is made up of one or more Sessions. See Manage Sessions for more information including how Sessions are managed and marked.

There are a couple of ways to add a new Learning Event. The primary method is via Manage Learning > Manage Sessions > Add Learning Event; alternatively via Manage Learning > Manage Learning Assets > Add Learning Event or the Add Learning wizard. After you create the Learning Event you add Sessions to it.
