Add Learning Asset - Learning Event

Add a Learning Event by following the instructions below and saving the Learning Event. You then come back and edit it to add Sessions. To add a Learning Event select Manage Learning > Manage Learning Assets and then click Add Learning Asset. Select New Learning Event from the options offered. You are then taken to the Manage Catalogues section of the CLS to create a new Learning Event. Note that you cannot get here directly from Manage Catalogues.

  1. Enter a Name and Identifier for the Learning Event
  2. Enter a Description if desired
  3. Select the Type of Learning Event from the drop-down box

  1. If you click the box next to Allow users to register interest two checkboxes appear and you must select one of them. The second option allows you to gauge User interest but it does not enrol them
  • Add interested users to the event waitlist:

  • Add interested users to the list without enrolling them:


Warning: Choose this option if you only want to gauge User interest in the event. It could potentially frustrate Users who do want to enrol.

  1. If you click the box next to Allow users to withdraw themselves from a session they are attending three checkboxes appear and you must select one of them:
  • When user withdraws keep them on the event waitlist for future sessions:

  • When user withdraws don't add them to the event waitlist for future sessions:

  • When user withdraws present them with the above options:

  1. Click the arrow next to Who is facilitating this event? Click and select a name from the drop-down box and click the plus icon. You can do this multiple times. You remove a name from the list by clicking the red minus icon at its right. Note that facilitating an event is different to facilitating a Session as this person can add/remove attendees to/from each Session.
  2. Click the arrow next to Competencies if desired. Two checkboxes appear and you must select one of them:
  • Automatically award all competencies when learner attends a session:

  • Require the facilitator to manually award the competencies:

  1. Click and select a Competency or Competency Set from the drop-down box and click the plus icon. You can do this multiple times. You remove an entry from the list by clicking the red minus icon at its right
  2. Click Save Learning Event

This takes you to the inspection screen where you can click Add Session if you want to add the Session details at this time.
