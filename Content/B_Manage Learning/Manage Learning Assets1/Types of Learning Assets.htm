Types of Learning Assets

There are many types of learning in the CLS system with associated icons. Depending on what has been configured with the plug-ins for your system, these include:

Janison Course

A container for learning content and Tests; created within the CLS Design section

Test

A container for Questions; can be assessable or non-assessable; created within the CLS Design section

Assessment Event

A Test that takes place at a specific time and place

Discussion Forum

A mechanism for Users to communicate with each other and receive feedback

Learning Event

A container for one or more Sessions where a User attends a venue (from Face to Face event to webinars)

Other Learning

External learning already completed by the User but acknowledged by the CLS system

SCORM course

A course created in a 3rd party application (such as Lectora and Captivate) that runs and is monitored by the CLS

Trackable document

A document used to deliver learning material (in PDF, Audio, or Video formats) that is monitored within the CLS

Assignment

A mechanism for learners to make file-based submissions and for markers to assess them

Survey

A tool for gathering information and feedback from Users

Plan

A mechanism for scheduling learning for a User after their analysis of their needs
