Add Learning Asset - Add Session

Sessions are part of Learning Events. In a Session a User actually attends a venue, including a virtual venue (webinar). Session information includes the when, where and capacity of the event. See Manage Sessions for more information about Sessions including how to create them, how they are managed and how they are marked.

There are a couple of ways to add a Session, each involves first specifying the Learning Event it will be attached to. The primary method is via Manage Learning > Manage Sessions > Add Session; alternatively via Manage Learning > Manage Learning Assets > Add Session or the Add Learning wizard. Regardless of the method used, certain fields need to be filled in and full details are found at Add Session.
