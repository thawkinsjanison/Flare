Add Learning Asset - Add Session

Sessions are where a User actually attends a venue, including a virtual venue (webinar). Session information includes the when, the where and the capacity of the event. This information can change from Session to Session within one Learning Event. See Manage Sessions for more details about Sessions including how they are managed and marked.

There are a couple of ways to add a Session, each involves specifying the Learning Event it will be attached to. You can select Manage Learning > Manage Sessions > Add Session, see Manage Sessions > Add Session

Alternatively, select Manage Learning > Manage Learning Assets. Click the desired Learning Event to inspect it and click Add Session.

You will be taken to the Manage Catalogue section of the CLS
