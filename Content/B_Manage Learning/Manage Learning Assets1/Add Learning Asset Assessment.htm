Add Learning Asset - Assessment Event

An Assessment Event is a scheduled instance of an assessment Test, i.e. a collection of Questions that are used for assessment.

To add an Assessment Event select Manage Learning > Manage Learning Assets and then click Add Learning Asset. Select New Assessment Event from the options offered.

  1. Type a Name and Identifier
  2. Type a Description if desired
  3. Select the Test Forms section to add Tests to this Assessment Event. Click to select an existing Assessment test and then click Add. You can repeat this multiple times
  4. Click and select from the options for the Test Form Allocation
  5. Enter the duration
  6. Use the date and time picker to enter the Start date and time
  7. If learners are permitted to start before the official Start time enter the maximum value for this in Pre-start duration
  8. If learners are permitted to finish after the official End time enter the maximum value for this in Start window
  9. Select Save Assessment Event
