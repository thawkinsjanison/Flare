Import Competencies

Rather than entering Competencies into the CLS system one at a time using Add a Competency, you can import a batch of Competencies using a spreadsheet.

To do this, select Manage Learning > Manage Competencies and select Import Competencies from the Actions dropdown.

  1. Download the default spreadsheet template, populate it with your data and save it on your local storage. Alternatively, create your own spreadsheet with the same columns as the template.
  2. The default template has a column labelled Linked Set Id. If you check the Add competency set requirement check box, a new Competency Set will be created for any values in that column in your spreadsheet that are not found in the CLS system.
  3. Select an email address where the import summary will be sent.
  4. Locate and select your spreadsheet from your local storage.
  5. Click Import and email result.
