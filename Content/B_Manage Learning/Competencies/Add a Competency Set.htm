Add a Competency Set

Sometimes you need to create a set of Competencies, such as an in-house collection of competencies. These are called Competency Sets.

Notes

  • A Competency must have already been created before it can be added to a Competency Set. Refer to Add a Competency for instructions on how to create Competencies.
  • Competency Sets have a defined Set Type. You can choose from the default Set Types and any other Set Types that have been created in advance. See Manage a Competency Set Type for instructions.

To add a Competency Set, select Manage Learning > Manage Competencies and select Add Competency Set from the Actions dropdown.

  1. Select a Competency Set Type from the dropdown.
  2. Add a Name; the Identifier will be entered automatically.
  3. Enter a Description if desired.
  4. Check the Active box to create an active Competency Set.
  5. Check Nationally Recognised box if applicable.
  6. Select the Competencies collapsible section to open it.
  7. Select a Competency from the dropdown and select the + icon to add it to the set. It is added to the list of Competencies.

    8. You can do this multiple times.

  8. Select Save Competency.
