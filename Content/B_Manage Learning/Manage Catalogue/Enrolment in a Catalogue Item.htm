Enrolment in a Catalogue Item

There are a number of ways that User enrolment in a Catalogue Item can take place. Broadly speaking, Catalogue Items can be configured so that learner Users are enrolled by authorised Users (such as trainers or administrators) or so that learner Users can enrol themselves.

To create User enrolments in Catalogue Items:

  • You can use the setup wizard Assign Learning
  • You can enrol Users in a Catalogue Item through Manage Catalogues. This technique brings you to the Enrolments section with that Catalogue Item, see Assign Learning â€“ Catalogue Item.
  • You can import upload a spreadsheet of Enrolment data containing lists of Catalogue Items, Users, Status, etc. See Import Learning History
  • You can create and run User Enrolment Rules where you select specific Users and automatically enrol them in specific Catalogue Items, see User Enrolment Rules

Learner Users can enrol themselves in Catalogue Items depending on the Self Enrolment Settings. These specify:

  • whether User self-enrolment is possible at all
  • the default enrolment method for all Catalogue Items
  • whether the default method can be overridden for an individual Catalogue Item, and
  • the range of enrolment methods possible for that Catalogue Item

See Settings > Self enrolment for more details of setting this up.

See Enrolments for more general information about managing enrolments once they have been created.
