Add Learning Wizard

Manage Catalogues offers a link to the Add Learning wizard .This is a setup assistant that offers a series of dialog boxes to complete the steps required to create a learning asset in the CLS and then link it to a new Catalogue Item with the same name, so that Users can enrol in it. See Add Learning for details

To launch the Add Learning wizard from Manage Catalogues

  • Select Manage Learning > Manage Catalogue
  • Click Add Learning Wizard
  • Note that you could also click Add Learning at the left hand side
