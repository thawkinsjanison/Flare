Inspect Catalogue Item

Inspecting a Catalogue Item gives you access to important information about it and allows you to edit some of this. The screen also offers the opportunity to edit more details about the Catalogue Item via selecting Edit Catalogue Item from the Actions drop-down menu, see Edit Catalogue Item

To inspect a Catalogue Item select Manage Learning > Manage Catalogue and click the desired Catalogue Item. The inspection screen is then displayed.

  1. Click Tasks and Notes to update them if desired and click Attachments to upload an Attachment
  2. Data about enrolled Users is displayed:
  • Enrolled Users in the Catalogue Item
  • Users who have Not Attempted the Catalogue Item
  • Users who have not completed the Catalogue Item (Incomplete)
  • Users who have Completed the Catalogue Item

Each piece of data is a link to the Enrolments section of the CLS where the appropriate selection of Enrolments in the Catalogue Item will be listed. If desired these can subsequently be modified (see Enrolment>View More about an Enrolment)

  1. The status of the Catalogue Item. Click Change to change it.

Make Private removes the item from the catalogue, i.e. hides it from learners although they can still be enrolled in it. Make Public adds the item to the catalogue, i.e. makes it active and visible to learners. Archiving the item takes it out of activity altogether

  1. The Learning assets linked to this Catalogue Item are listed in this screen. The list can be edited in Edit Catalogue Item
  2. Click the arrow at the left of Tags to see the linked Tags and click the pencil icon to link new Tags
  3. Click the arrow next to Competencies to view the linked Competencies
  4. Click the arrow at the left of Recent Activities to view recent activities

Note the Instalaunch link next to the Name and Identifier. If self-enrol has been set for this Catalogue Item (see Add Catalogue Item) then this address will allow a guest enrolment in the Catalogue Item.
