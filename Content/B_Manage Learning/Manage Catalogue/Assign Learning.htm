Assign Learning

Assign Learning is a wizard, i.e. a setup assistant, which offers authorised Users a series of dialog boxes to complete the steps required to enrol Users in Catalogue Item(s) and schedule the enrolment process. It is available through Manage Learning > Assign Learning

Manage Catalogues provides a separate link to this wizard through a Catalogue Item. The wizard starts off with that Catalogue Item selected as part of the Content that User(s) enrol in.

To run this Add Learning Wizard

  • Select Manage Learning > Manage Catalogue
  • Click the desired Catalogue Item
  • Select Assign Learning from the Actions drop-down menu.

This launches at Step 1 of the Assign Learning wizard, with the Catalogue Item already added. You can add further Content if desired, then proceed with the next steps of the wizard. See Assign Learning for details
