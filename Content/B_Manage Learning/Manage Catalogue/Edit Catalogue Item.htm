Edit Catalogue Item

To edit a Catalogue Item, Select Manage Learning > Manage Catalogue. Locate the desired Catalogue Item and click it. This allows you to inspect it, change its Status and edit its linked Tags, see Inspect Catalogue Item.

To further edit the Catalogue Item, select Edit Catalogue Item from the Actions drop-down menu.

The subsequent screen offers editing of all the same fields that are present when you create a Catalogue Item, see Add Catalogue Item for details of the fields involved.

You can also delete the Catalogue on this screen, but it is recommended that Catalogue Items be archived rather than deleted, see Archive a Catalogue Item for details
