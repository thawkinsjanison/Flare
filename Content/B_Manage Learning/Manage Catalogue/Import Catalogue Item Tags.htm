Import Catalogue Item Tags and link to Catalogue Items in bulk

Tags can be assigned to existing Catalogue Items in bulk

Select Manage Learning > Manage Catalogue and click Import Catalogue Item Tags

  • Download the Excel Template and complete the required details. The Supported Attributes sheet in this document gives more details about the fields required
  • Save the file.
  • Click the check boxes if you want to import new Tags and/or Tag Types as part of this import
  • Select your saved Excel file, enter an email address, then click Import and email result.
