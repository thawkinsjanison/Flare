Manage Catalogue

Catalogue management is reached by selecting Manage Learning > Manage Catalogue. The screen displays all Catalogue Items. This list can be filtered by the Search function (see Search for a Catalogue Item)

The Catalogue Items are divided into the following tabs:

  • Available in Catalogue - the Catalogue Item is Public - students can see it in Browse Learning (and also self-enrol if that’s enabled)
  • Not in Catalogue – the Catalogue Item is Private and students can’t see it in Browse Learning (but they can be enrolled in it by an authorised User)
  • Draft - the Catalogue Item is incomplete – e.g. the Add learning wizard was started but not completed
  • Archived - the Catalogue Item has been Archived and is unavailable for all kinds of enrolments
  • All

Click the tabs to see the items in each category.

On this screen you can view the creation date for each Catalogue Item. The number under Number of Enrolments is also a link that takes you to the Enrolments section of the CLS with a list of all the enrolments for that Catalogue Item. You can also continue from this point to add new User enrolments to the Catalogue Item.

Note that the icons displayed under the Type heading indicate the type of Learning Asset contained in each Catalogue item, see Types of Learning Asset. Where there is more than one Learning Asset, the Catalogue Item is technically a Program and displays the icon for a Program.
