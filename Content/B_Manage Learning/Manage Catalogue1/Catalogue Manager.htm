Catalogue Manager

To access the Catalogue Manager browse to Manage Learning > Manage Catalogue.

The Catalogue Items are filtered into the following tabs:

  • Available in Catalogue - the Catalogue Item is Public and learners can see it in Browse Learning. Setting a Catalogue Item to Make Public will add it to this tab.
  • Not in Catalogue – the Catalogue Item is Private and learners can’t see it in Browse Learning. They can still enrolled in it by an authorised User. Setting a catalogue to Make Private will add it to this tab.
  • Draft - the Catalogue Item is incomplete – e.g. the Add learning wizard was started but not completed.
  • Archived - the Catalogue Item has been Archived and is unavailable for enrolments. Setting a catalogue to Archive will add it to this tab.
  • All

Manage Catalogue Statuses in the Catalogue Manager

The status of one or more Catalogue Items can be set by:

  1. Selecting one or more Catalogue Items in the left column.
  2. Selecting one of the buttons at the bottom of the list, i.e. Archive.
