Approve Learning Requests

Select Manage Learning > Approve Learning

  • Select the required tab from the four at the top of the screen.  In the example below we have selected Requested.
  • Select the check box to the left of the user name and click Approve or Reject.

  • You will then need to check Yes to complete the process.

You can now see that one user has been approved from the screenshot below.  If you click on the Approved tab your user information will load.
