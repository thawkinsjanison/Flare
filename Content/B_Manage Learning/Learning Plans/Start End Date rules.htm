Start / End Date rules

When you first open this section, you may be confused that it seems to have nothing you can do in it. Remember to use the pencil to edit.

When you do, you can check “Allow Start/End date”. This opens up all the options to you.

Some organisations want learning plans to have specific start / end dates. Some prefer to allow learners to choose their own start / end dates. Others don’t want plans to have end dates at all.

If you don’t want start / end dates on your plans, don’t check “Allow Start / End date”

If you want the learner to control their own start / end dates, check “Allow Start / End Date” and “Allow Learner / Manager to control Dates” but don’t set any rules

If you want more control over learning plans, you need to use start and end date rules

Start date rule

There are three choices

  1. The plan can start on the day the learner is enrolled in it
  2. “Day of current calendar year” - the plan will start on a particular day of the current year (even if that day is in the past)
  3. “Day of year from today” - the plan will start on a particular date when it next occurs. This means that, for example, if today is 15th March 2015 and you select 1 May, the plan will start on 1 May 2015. However, if you choose 1 March, the plan won’t start until 1 March 2016.

End date rule

There is only one choice – how many days or months after the plan starts should it end.

Be careful! These rules only apply at the time the learner is enrolled in the plan. Once they are enrolled, any changes you make here won’t affect their plans.
