Manage CPD progress

Manage CPD Progress screen is made up of graphs depicting the overall CPD progress.

Browse to Manage Learning > Manage CPD progress.

The Manage

In the ‘Details’ section there are two options and several tabs:

  1. Timeline – This displays a list of users with the following columns:
  • User
  • Plan
  • Timeline (Shows monthly status)
  • CPD Expiry
  • Status
  • Actions > View More

  1. Points’ – This displays a list of users with the following columns:
  • User
  • Plan
  • Tags (Shows tag status)
  • CPD Expiry
  • Status
  • Actions > View More

  • Each of the options above contains a series of Tabs you can select including:
  • All Users
  • Non-Compliant Users
  • High Risk Users
  • Low Risk Users
  • Compliant Users
  • Click View More next to the required user to inspect that users CPD plan membership
