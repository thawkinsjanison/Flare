Mark a Session

Authorised User(s) need to review the Users attending a Session and to mark whether they attended or not. Depending on what has been set up for the Learning Event, Competencies may also need to be awarded.

To mark a Session select Manage Learning > Manage Sessions. Locate the Session and click its Mark icon.

The main components of the screen show:

  1. The name of the Session and a hyperlink to its Learning Event
  2. The details of the Session
  3. A list of the Attendees.

  1. To mark an attendee, click the Status drop-down list for each User and select one of the attendance options.

    2. If Enable 'Interviewed Or Rpl' Attendance Status has been checked in the Learning Event Settings, then you can also select this status option. In this case you must also add some notes, see 5.

  2. You can add notes to a User's attendance record by clicking the pencil icon. A box opens for text entry. Click the pencil again to close it.
  3. Alternatively, if there are a large number of Users, you can click the checkbox at the top left of the list or the box to the left of each User and use the Mark Selected As option to bulk assign attendance status. 
  4. Marks are not submitted until you click Submit Marks.
