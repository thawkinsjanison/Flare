Session list and summary screens

Each Learning Event Session has its own summary screen that learners can view in the CLS.

From this page, learners can:

  • View Session information.
  • Download attachments.
  • Choose whether to attend or withdraw from the session (if permitted, see Sessions process for details)
  • See a Google map popup of the venue Location by clicking the Venue name:

Note that Show map needs to be checked for the Session and its Location and Venue set up with an address, see Locations and Venues for details

Learners can access this screen from the Sessions list screen that they see after they enrol in the Learning Event or from their My Learning screen. Note that what they see here and how they get to it depends on the intersection of various settings, see Sessions process.

Note also that this Session list screen is searchable, based on the Venue's name, suburb or postcode and start dates.

Another access to the Summary screen is provided to learners when the administrator inserts a hyperlink in the email message that is sent to Users when they enrol in a Session. This link takes learners to the Session summary screen.

To insert this hyperlink using the rich text editor, select Insert > insert link or click the link icon :

Type the name of the token [Session.SessionDetailsLink] into the URL address:

In the example above, the link will appear as Click here to learners and it will take them to the Session summary screen.

This link to the Session summary screen can also be inserted into any Template relating to Learning Event Sessions by using the same token, see Message Templates.
