Manage Sessions

To manage and mark Sessions select Manage Learning > Manage Sessions

Mange Sessions is a kind of one stop shop for Sessions. It displays all Sessions and provides links to all the things you need to do with then. The screen divides Sessions into three categories: Published, Completed and Setup.

The main aspects of the page are:

  1. Click Search for Learning Events Sessions to search the list.

    2. You can enter search terms including: the Learning Event name (first row); Venue and Location (second row); and dates (third row).

    Enter search terms and click Search.

    If you have checked Enable advanced session location search in the Learning Event Settings then the second row can also be searched by state, postcode or suburb. This may run more slowly when there are a large number of Sessions.

  1. Published – Click this tab to see Sessions that have been published into the Catalogue.
  2. Completed – Click this tab to see Sessions that have been closed off by the administrator at the end of marking.
  3. Setup – Click this tab to see Sessions that are still in setup mode i.e. they have not been published to the Catalogue.
  4. Each Session is listed with its start date and time, Location, the Learning Event it belongs to and its attendee details.

    5. Clicking a heading will sort the list in ascending or descending order based on this value.

  1. Click the eye icon to inspect a Session in Manage Learning Assets.
  2. Click the pencil icon to edit a Session in Manage Learning Assets.
  3. Click the people icon to manage attendees for a Session and move Users to and from waiting lists.
  4. Click the tick icon to mark the Users enrolled for a Session.
  5. Click Add Learning Event to add a new Learning Event in Manage Learning Assets.
  6. Click Add Session to add a new Session in Manage Learning Assets.

If you click you will see a popup item with further details about the Venue and Location.
