Learning Event Settings

These are specified via Settings > Learning Event. There are several settings that for Sessions and Learning Events:

  1. Group Type – click and select a Group type that will display on the Session marking screen.
  2. Interviewed or RPL can be added to the options that a Session marker can choose from. If the option is selected then it is compulsory for the marker to add some notes.
  3. These options enable the movement of whole groups and org units between waiting lists and attendee lists.
  4. Check this option to display outstanding Sessions on a learner User’s calendar.

  1. Check this option to enable more powerful searching at Manage Sesssions and Assign Learning. Text searches will also match against the Venue and Location names, town, postcode and state. Note that with a very large number of Sessions this may make the search run a little slower.
  2. If a number is entered here, attendees will receive a reminder notification via email. This will be sent a number of minutes before the Sesssion starts; you specify this number.
  3. Select a template from the drop-down list. This is the template that will be used when printing an attendees list.
  4. Click Save Learning Event Settings.
