RPL

Recognition of Prior Learning (RPL) allows an administrator to mark a User as competent or compliant if the User has achieved the requirements through an external provider or in some other manner. Using this method a completed enrolment is created for the User in a specified Catalogue Item.

Note that RPL is not enabled by default. To enable it select Settings > Tenant Settings > Miscellaneous > Enable RPL

To create RPL for a User against a Catalogue Item, select Manage Learning > Enrolments and then click RPL.

This takes you to the Assessment Events section of the CLS. Complete the details as required:

  • Catalogue Item
  • User
  • Completed Date
  • Points (if enabled)
  • Comments - if the prior learning has been completed externally you need to enter details in this box
  • Click Save RPL
