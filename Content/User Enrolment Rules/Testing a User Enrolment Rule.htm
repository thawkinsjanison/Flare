Testing a User Enrolment Rule

To ensure that you have the rule set up correctly and also to check who will be affected by the rule a Test feature is available.

Browse to Settings > User Enrolment Rules.

  1. Find and Select an Enrolment Rule.

  1. Select Test Action Rule.

The Enrolment Rule’s testing page contains three tabs.

  1. The To Be Enrolled tab shows the users who will be enrolled when the rule is run.
  2. The Already Enrolled tab shows the users who have already been enrolled by this rule.
  3. The All tab show both the users to be enrolled and those users already enrolled.

If the list of user in the To Be Enrolled tab do not match the users you would like to, it is a good idea to return to the User Enrolment Rule and adjust the filter options before continuing.

  1. Select Run Now.

After selecting Run Now, the users in the To Be Enrolled tab will be enrolled into the courses and moved to the Already Enrolled tab.
